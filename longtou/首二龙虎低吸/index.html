首板+二板龙虎榜低吸策略

1. 选股条件

  • 选股范围：全市场A股（主板为主）
  • 排除范围：
  • 科创板、北交所、创业板、B股
  • ST股及其他退市标签股
  • 当日停牌、涨跌停股票

核心筛选标准：

  • 龙虎榜要求：
  • 前日登上龙虎榜
  • 机构专用席位净买入 > 0（优先选择，非必须）
  • 买入前五合计金额 > 5000万元（保证资金力度）
  • 涨停要求：
  • 首板或二板涨停
  • 昨日收盘价等于涨停价
  • 成交量要求：
  • 昨日成交量 > 前5日平均成交量的 2倍以上（简化计算）
  • 不要求精确4–6倍，只需“明显放量”

附加筛选条件：

  • 概念热度：所属概念板块当日至少有2只以上涨停股（同花顺“概念板块”可查）
  • 换手率：换手率 < 50%（排除庄股）
  • 流通市值：50亿 ～ 800亿元（兼顾流动性与弹性）

2. 买入条件

  • 交易频率：低频择机（平均每月1–3次，非每日交易）
  • 买入规则：
  • 放弃高开追涨：开盘涨幅 > 5% 则放弃买入；
  • 只做回调或横盘：当日涨幅在 -2% 至 +3% 之间；
  • 最多持有1只股票；
  • 单票仓位 ≤ 20%（不再全仓）；
  • 买入方式：
  • 尾盘14:45–14:55市价单买入（避开早盘情绪干扰）
  • 按100股整数倍下单
  • 执行时间：14:45后（非9:26）

3. 卖出条件

  • 止损卖出：
  • 从买入成本起算，亏损 ≥ 7% 立即卖出；
  • 若买入后次日低开低走无反抽，可提前止损。
  • 卖出时机：仅设两个时间点
  • 14:50（尾盘决策）
  • 次日冲高（如+5%以上可部分止盈）
  • 卖出逻辑：
  • 持有满3个交易日仍未上涨，第4日开盘卖出；
  • 若期间再次涨停，次日冲高卖出；
  • 不再“不涨停就卖”，给予合理波动空间。

4. 仓位管理

  • 持仓数量：最多1只股票（保持专注）
  • 资金分配：单只股票仓位 10%～20%，其余资金持有货币ETF（511880）
  • 调仓频率：仅当出现高确定性机会时替换，不每日调仓
  • 风险控制：
  • 不满仓、不杠杆、不All in；
  • 同一概念板块3日内最多操作1只。

5. 风控管理

  • 流动性风控：仅参与日成交额 ≥ 2亿元的股票
  • 质量风控：排除ST、次新、高质押、高商誉问题股
  • 集中度风控：单票≤20%，杜绝全仓赌博
  • 止损风控：7%硬止损，不可协商
  • 概念风控：必须有板块效应（至少2只同概念涨停）
  • 情绪风控：若市场最高连板 ≤ 2板，暂停新开仓

6. 龙虎榜逻辑（简化应用）

  • 机构认可：优先选择“机构专用席位”上榜的股票（同花顺龙虎榜标注“机构”）
  • 买入力度：买入金额 > 5000万视为有效信号
  • 净流入：净买入为正即可，不强求占比
  • 持续性：首板/二板代表启动初期，但只做分歧低吸，不做启动追高

7. 手动实操步骤（纯人工交易版）

第一步：盘后筛选（前一交易日20:00后）

  1. 打开同花顺 → 搜索“龙虎榜” → 查看“营业部买入前五”
  2. 筛选：
  3. 非ST、主板、昨日涨停
  4. 买入金额 > 5000万
  5. 所属概念有≥2只涨停股（点进概念页验证）
  6. 将候选股加入自选股

第二步：次日观察（9:30–14:30）

  1. 观察候选股：
  2. 是否高开 ≤ 5%？
  3. 是否在-2%至+3%区间震荡？
  4. 板块是否仍有热度？
  5. 若高开 > 5% 或快速冲板，直接放弃

第三步：尾盘买入（14:45–14:55）

  1. 确认当日涨幅在 -2% 至 +3%；
  2. 确认未涨停、未跌停；
  3. 手动市价单买入（仓位≤20%）；
  4. 设置7%止损条件单（买入价×0.93）

第四步：持仓管理（买入后）

  • 第2日：若冲高+5%以上，可卖半仓；
  • 第3–4日：未达目标则第4日离场；
  • 任何时点破7%止损，立即卖出。

第五步：纪律维护

  • 每月最多操作3次；
  • 龙虎榜股失败后，3日内不碰同板块；
  • 宁可错过，不可做错。
