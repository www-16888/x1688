连板龙头策略

1. 选股条件

✅ 选股范围

  • 全市场A股（主板为主）

❌ 排除股票（同花顺“条件选股”中勾选）

  • ST/*ST/退市股
  • 科创板（688开头）
  • 北交所（8开头）
  • 停牌股
  • 上市未满50天的新股

🔍 技术信号（问财语句实现）

非ST，非科创板，非北交所，主板，昨日涨停，连续涨停天数大于等于2天，5日平均换手率小于10%，流通市值大于50亿，所属行业近一月涨幅排名前30%

✅ 说明： - 昨日涨停：确认昨日涨停； - 连续涨停天数大于等于2天：筛选连板龙头； - 5日平均换手率小于10%：避免流动性过高的高波动股； - 流通市值大于50亿：保证流动性和机构关注度； - 所属行业近一月涨幅排名前30%：提升成功率的关键过滤。

2. 买入条件

  • 买入时机：上午9:30开盘后立即执行
  • 买入前提：
  • T日满足上述技术信号
  • T+1日开盘时股价未涨停（若涨停则限价单排板）
  • 仓位控制：单只 ≤ 10% 总资金
  • 最大持仓：同时持有 ≤ 10只
  • 买入方式：
  • 未涨停：使用市价单立即买入
  • 已涨停：使用限价单排板买入

⚠️ 关键改进：开盘即行动，减少犹豫时间，提高命中率。

3. 卖出条件

情况 操作
卖出条件1 当日不涨停
卖出条件2 持有超过2个交易日
卖出条件3 已实现盈利
执行时点 下午14:50执行卖出
卖出原则 满足条件1的前提下，条件2或条件3任一满足即卖出

✅ 不再依赖复杂加减仓规则，改用明确的卖出条件，大幅降低操作难度。

4. 仓位与风控

项目 规则
单票仓位 ≤ 10%
最大持仓 10只
空仓条件 上证指数 < MA20 或 市场连板高度 ≤ 2板
流动性要求 日成交额 ≥ 3亿元（F10可见）
组合风控 账户总回撤 ≥ 5%，暂停开仓1周

5. 手动实操步骤

🕒 第一步：盘前准备（每个交易日9:01前）

  1. 打开 同花顺 → i问财
  2. 输入以下语句并点击“选股”： text 非ST，非科创板，非北交所，主板，昨日涨停，连续涨停天数大于等于2天，5日平均换手率小于10%，流通市值大于50亿，所属行业近一月涨幅排名前30%
  3. 将符合的加入自选股（最多10只）

🕒 第二步：开盘决策（9:15-9:30）

  1. 查看自选股：
  2. 是否仍在涨停？
  3. 成交量是否正常？
  4. 所属板块是否有持续热度？
  5. 选1–2只最强标的，市价买入 ≤10%仓位

🕒 第三步：买入执行（9:30）

  1. 开盘未涨停：立即市价买入
  2. 开盘涨停：限价单排板买入
  3. 计算单股买入金额：总资金 / 10

🕒 第四步：盘中监控

  1. 持仓跟踪：
  2. 记录每只股票的买入日期
  3. 监控是否继续涨停
  4. 盈利计算：
  5. 实时计算持仓盈亏
  6. 为卖出决策做准备

🕒 第五步：卖出执行（14:50）

  1. 卖出条件检查：
  2. 检查持仓股是否涨停
  3. 计算持有天数
  4. 计算当前盈亏
  5. 卖出决策：
  6. 不涨停的股票进入卖出流程
  7. 持有超2天或已盈利的股票卖出
  8. 严格执行卖出纪律

🕒 第六步：盘后复盘

  1. 持仓整理：
  2. 更新持仓记录
  3. 计算总体收益
  4. 准备次日交易计划
  5. 策略优化：
  6. 分析交易记录
  7. 调整选股参数
  8. 优化买卖时机
