高股息低估值成长策略（散户实操优化版 · A股合规版）

1. 策略分类和核心逻辑

  • 一级分类：价值投资策略
  • 二级分类：高股息 + 低估值 + 稳健成长
  • 核心逻辑：通过同花顺问财自然语言一键筛选股息率高、估值低、盈利稳、有成长的主板优质股，每季度调仓一次，避免频繁交易。在获取稳定股息的同时，分享企业稳健增长的红利。策略完全基于免费软件可获取数据，符合A股T+1、100股整数倍、交易时段等规则，适合普通散户每季度花15分钟操作。

2. 选股条件

  • 选股范围：A股主板（60/00开头）
  • 排除范围：
  • 创业板（300）、科创板（688/689）、北交所（4/8开头）
  • ST、*ST、退市风险股
  • 停牌股

  • 上市不满300天的新股

  • 可执行筛选条件（全部问财支持）：

  • 股息率：近12个月股息率 > 4%
  • 估值水平：市盈率（PE-TTM）在 5–25 之间（排除亏损和超高估值）
  • 盈利能力：净资产收益率（ROE） > 8%（保障盈利质量）
  • 成长能力：近一年净利润同比增长 > 10%（体现持续成长）
  • 流动性保障：近20日日均成交额 > 5000万元

✅ 同花顺问财语句（每季度第一个交易日9:00直接复制）：
主板股票，非ST，上市天数大于300，股息率大于4，市盈率大于5且小于25，ROE大于8，净利润同比增长大于10，近20日日均成交额大于5000万

3. 买入条件

  • 调仓频率：每季度一次（1月、4月、7月、10月的第一个交易日）
  • 买入时点：9:35–10:00（避开集合竞价波动）
  • 买入规则：
  • 先卖出上季度持仓中未进入新前8名的股票；
  • 再用可用资金等分买入新入选的8只股票；
  • 若结果不足8只，有多少买多少，不强行凑数。
  • 数量限制：最多持有8只股票（原10只，降低操作难度）
  • 排序标准：问财结果按“股息率”从高到低，取前8只

4. 卖出条件

  • 定期调仓卖出：
  • 每季度调仓日，卖出不在新选股前8名的持仓股；
  • 保留仍在前8名的股票（减少交易频率和成本）。
  • 特殊情况卖出：
  • 股票被ST或停牌 → 次日开盘立即卖出；
  • 连续两个季度未分红 → 下次调仓时剔除。
  • 取消“涨停打开卖出”规则：
  • 高股息股极少涨停，此规则无实际意义，避免无谓盯盘。

✅ 完全符合T+1：所有卖出均为上季度或更早买入的股票，可当日卖出。

5. 仓位管理

  • 持仓数量：5–8只（根据筛选结果动态调整）
  • 资金分配：等权重，每只约占12.5%–20%资金
  • 总仓位：
  • 若选出≥5只 → 满仓运作；
  • 若选出<5只 → 剩余资金持有货币ETF（511880），避免空仓踏空或满仓劣质股。
  • 权重分配：手动计算每只买入金额，确保大致等权（允许±5%误差）

6. 风控管理

  • 估值风控：PE 5–25区间，排除亏损股和泡沫股；
  • 质量风控：ROE > 8% + 净利润增长 > 10%，确保“真成长+真盈利”；
  • 股息风控：股息率 > 4% + 近20日成交 > 5000万，避免“股息陷阱”；
  • 分散风控：8只股票天然分散行业（银行、电力、高速、煤炭等高股息板块）；
  • 纪律风控：仅季度调仓，杜绝情绪化交易。

7. 手动实操步骤（纯人工交易版）

第一步：季度选股（每季度第一个交易日 9:00–9:20）
1. 打开同花顺问财，粘贴上述问财语句；
2. 点击“问一下”，等待结果；
3. 在结果页点击“股息率”列，从高到低排序；
4. 取前8只，记录代码、名称、股息率、PE、ROE。

第二步：调仓准备（9:20–9:35）
1. 对比上季度持仓与新名单；
2. 标记需卖出的股票（不在前8名者）；
3. 计算可用资金，除以新买入数量，得出每只目标金额。

第三步：调仓执行（9:35–10:00）
1. 卖出：逐笔卖出需剔除的旧股（确保未涨停）；
2. 买入：对新入选股票，按目标金额下单（数量=金额÷当前价，取100股整数倍）；
3. 若资金有余，买入 511880 货币ETF。

第四步：盘中监控（无需盯盘）
- 不设盘中操作，高股息策略以持有为主。

第五步：持仓管理（季度间）
- 每月查看一次持仓股是否发布分红预案；
- 若某股被ST或长期无成交，提前标记下季度剔除。

第六步：纪律执行
- 严格按季度调仓，不因短期波动加减仓；
- 不追热点、不炒题材，坚守高股息价值逻辑。

交易成本优化：
- 季度调仓 → 年交易仅4次，大幅降低佣金+印花税；
- 单只持仓≥500股 → 避免“最低5元佣金”侵蚀收益；
- 不交易涨停股 → 无滑点风险。

策略优势：
- 问财一键筛选，无需计算股息率、PEG、Beta；
- 季度调仓，每月仅需15分钟，适合上班族；
- 高股息提供安全垫，熊市抗跌，牛市跟涨；
- 完全符合A股规则，无T+1冲突，无最小单位违规。

风险提示：
- 高股息板块（如银行、电力）可能阶段性跑输成长股；
- 股息率受股价波动影响（股价涨→股息率降）；
- 建议首次使用≤20%资金试运行1个季度；
- 若遇极端熊市（如沪深300单季跌>15%），可临时将仓位降至50%，其余持511880。
