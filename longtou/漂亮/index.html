漂亮50行业龙头策略（散户实操优化版 · A股合规版）

1. 策略分类和核心逻辑

  • 一级分类：基本面量化策略
  • 二级分类：行业龙头 + 价值成长 + 月度轮动
  • 核心逻辑：在全市场主板股票中，通过同花顺问财自然语言一键筛选各行业高ROE、高增长、合理估值的龙头公司，每月调仓一次，构建5–8只分散化组合。策略放弃中证500成分限制、行业内百分比排名、PEG计算等复杂操作，全部条件均可在免费软件中实现，符合A股T+1、100股整数倍、交易时段等规则，适合普通散户每月操作15分钟。

2. 选股条件

  • 选股范围：A股主板（60/00开头）
  • 排除范围：
  • 创业板（300）、科创板（688/689）
  • ST、*ST、退市风险股
  • 停牌股

  • 上市不满250天的新股

  • 可执行筛选条件（全部问财支持）：

  • 盈利能力：净资产收益率（ROE） > 10%
  • 成长能力：近一年净利润同比增长 > 15%
  • 估值合理：市盈率（PE-TTM）在 10–30 之间
  • 龙头特征：总市值 > 300亿元
  • 流动性保障：近20日日均成交额 > 5000万元

✅ 同花顺问财语句（每月第一个交易日9:00直接复制）：
主板股票，非ST，上市天数大于250，ROE大于10，净利润同比增长大于15，市盈率大于10且小于30，总市值大于300亿，近20日日均成交额大于5000万

3. 核心筛选流程

第一步：基础过滤（问财自动完成）
- 通过问财语句自动排除ST、新股、小市值、低成交股票。

第二步：行业分散保障（人工辅助）
- 在问财结果页，点击“所属行业”列，观察是否覆盖≥5个不同申万一级行业；
- 若结果过度集中（如80%为银行），手动剔除同行业排名靠后者，保留各行业ROE最高1–2只。

第三步：龙头确认（自动+人工）
- 问财结果已按市值>300亿筛选，默认为行业龙头；
- 人工快速核对：是否为行业知名公司（如家电选美的/格力，白酒选茅台/五粮液等）。

第四步：最终排序
- 按ROE从高到低排序，取前8只；
- 若行业分散良好，可保留8只；若集中，取5–6只。

4. 买入条件

  • 调仓周期：每月一次（每月第一个交易日）
  • 持仓数量：5–8只股票（根据行业分散情况动态调整）
  • 资金分配：
  • 可用现金等分买入每只股票；
  • 每只买入数量为100股整数倍。
  • 执行时间：9:35–10:00（避开集合竞价，确保成交）

5. 卖出条件

  • 定期调仓卖出：
  • 每月调仓日，卖出不在新选股名单中的持仓股；
  • 保留仍在名单中的股票（减少交易频率）。
  • 止损卖出：
  • 收盘价连续2日低于20日均线 → 次日开盘卖出；
  • 替代原“5日高点回撤15%”（更简单、可盘后判断）。
  • 特殊情况卖出：
  • 股票被ST或停牌 → 次日开盘立即卖出。

✅ 完全符合T+1：所有卖出均为上月或更早买入的股票，可当日卖出。

6. 仓位管理

  • 持仓数量：5–8只（行业分散优先）
  • 资金分配：等权重，每只约占12.5%–20%资金
  • 总仓位：
  • 若选出≥5只 → 满仓运作；
  • 若选出<5只 → 剩余资金买入货币ETF（511880）。
  • 行业分散：强制覆盖≥5个不同行业（消费、制造、金融、医药、周期等）

7. 风控管理

  • 质量风控：ROE>10% + 净利润增长>15% + PE合理，保障“真龙头、真成长”；
  • 价格风控：20日均线止损，避免深度回撤；
  • 流动性风控：市值>300亿 + 日均成交>5000万，确保可买卖；
  • 分散风控：5–8只 + ≥5行业，避免单一风险；
  • 操作风控：月度调仓，年交易仅12次，成本可控。

8. 交易频率

  • 选股计算：每月第一个交易日9:00（问财10秒完成）
  • 买入执行：9:35–10:00
  • 止损监控：每日收盘后查看K线是否破20日均线（无需盯盘）
  • 调仓周期：每月1次（非10天）

9. 策略特点

基本面选股优势：
- ROE+净利润增长+PE三因子，兼顾质量、成长、估值；
- 市值>300亿，聚焦行业龙头；
- 问财一键筛选，无需编程或专业终端。

风险控制完善：
- 20日均线止损，简单有效；
- 行业强制分散，降低风格风险；
- 月度调仓，避免过度交易。

执行效率高：
- 每月15分钟完成；
- 无需维护成分股名单；
- 无需计算PEG或行业百分位。

10. 手动实操步骤

第一步：数据准备（9:00–9:10）
1. 打开同花顺问财，粘贴上述问财语句；
2. 点击“问一下”，导出结果（通常20–40只）。

第二步：行业分层（9:10–9:15）
1. 在结果页点击“所属行业”排序；
2. 人工剔除同行业排名靠后者，确保≥5个行业；
3. 按ROE从高到低取前8只（或5–6只）。

第三步：调仓执行（9:35–10:00）
1. 卖出上月持仓中未入选新名单的股票；
2. 等分资金买入新入选股票（100股整数倍）；
3. 余钱买入511880。

第四步：止损监控（每日收盘后）
- 打开持仓股K线，查看是否连续2日收盘价 < 20日均线；
- 若是，次日9:30卖出。

第五步：纪律执行
- 严格按月调仓，不因短期波动干预；
- 不追高、不炒小、不碰ST，坚守龙头逻辑。
