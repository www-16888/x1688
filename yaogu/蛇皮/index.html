蛇皮走位小市值策略（散户实操优化版）

1. 策略的风险等级、分类和策略核心逻辑

  • 风险等级：R3（稳健偏积极）——降低频率、简化条件、减少持仓
  • 策略分类：小市值策略 / 均值回归策略（简化版）
  • 策略核心逻辑：在市场相对平稳期，筛选低价、小市值、基本面不差、近期未大涨的股票，于次日早盘低吸，持有3–5天博弈小盘股轮动反弹。核心是“小市值+低价+非亏损+低位置”，避免复杂计算，聚焦可观察、可买入、可卖出的标的。

2. 选股条件（适配普通软件）

  • 选股范围：A股主板 + 创业板（排除科创板688/689、北交所4/8开头、ST/*ST、停牌股、上市<250天新股）

  • 筛选条件（全部可通过同花顺问财实现）：

  • 价格要求：股价 < 10元

  • 市值要求：流通市值 < 50亿元（确保小盘，但保留一定流动性）
  • 基本面底线：近4个季度无连续亏损（问财可近似为“非亏损股”）
  • 状态排除：非ST、非停牌、近5日涨幅 < 15%（避免追高）
  • 流动性保障：昨日成交额 > 2000万元

✅ 同花顺问财语句（盘后15:30使用）：
股价<10元，流通市值<50亿，非ST，主板或创业板，上市天数>250，昨日成交额>20000000，近5日涨幅<15%，非亏损股
（注：“非亏损股”为问财关键词，代表近一年未亏损；若结果过多，可加“换手率<8%”过滤）

✅ 通达信选股公式（盘后运行）：
```pascal { 蛇皮走位 - 散户优化版 } PRICE_LT_10 := C < 10; MCAP_LT_5B := FINANCE(40) < 5000000000; NOT_ST := NAMELIKE('ST')=0 AND NAMELIKE('*ST')=0; MAIN_OR_GEM := CODELIKE('60') OR CODELIKE('00') OR CODELIKE('30'); NOT_NEW := FINANCE(42) > 250; LIQ_YESTER := AMO > 20000000; NO_RECENT_RALLY := (C / REF(C,5) - 1) < 0.15; NOT_LOSS := FINANCE(33) > 0; { 净利润>0，近似非亏损 }

PRICE_LT_10 AND MCAP_LT_5B AND NOT_ST AND MAIN_OR_GEM AND NOT_NEW AND LIQ_YESTER AND NO_RECENT_RALLY AND NOT_LOSS; ```

3. 买入条件

  • 买入时机：次日（T+1日）9:35–10:00（避开开盘波动，确保流动性）
  • 前提条件：
  • 中证1000指数（000852）处于20日均线上方（代表小盘股环境不差）
  • 全市场跌停股 < 20家（无极端恐慌）
  • 买入方式：
  • 从问财结果中按流通市值从小到大排序，选前3–5只；
  • 等分资金买入；
  • 限价单：价格=前日收盘价±0.02元。

4. 卖出条件（简化+可执行）

  • 止盈卖出：
  • 盈利 ≥ 8% → 次日冲高卖出；

  • 持股第5日14:50 → 无论盈亏强制卖出。

  • 止损卖出：

  • 亏损 ≥ 5% → 当日或次日开盘止损；

  • 跌破10日均线 → 收盘前卖出。

  • 例外：若个股突发涨停，可持有至开板日卖出。

5. 仓位管理

  • 最大持仓数：3–5只（原10只不可行）
  • 单股仓位：≤ 总资金的25%
  • 总仓位：仅在市场环境平稳时操作，不满足前提时空仓
  • 换仓频率：每周调仓1次（如每周一选股，周二买入）

6. 风控管理

  • 市场风控：仅当中证1000站上20日线时操作
  • 个股风控：流通市值<50亿+成交额>2000万+非亏损，规避垃圾股
  • 时间风控：5日持有上限，防止阴跌
  • 亏损风控：单笔最大亏损5%
  • 流动性风控：早盘买入+非极小盘，确保可成交

7. 手动实操步骤（纯人工+普通软件版）

第一步：盘后选股（15:30–16:30）

  1. 打开同花顺问财，输入筛选语句；
  2. 导出结果，按“流通市值”排序；
  3. 选前5只，加入自选股分组“小市值观察”。

第二步：市场环境检查（次日9:30）

  1. 查中证1000（000852）是否站上20日均线；
  2. 问财查“今日跌停家数”是否 < 20；
  3. 若满足，准备买入。

第三步：买入执行（9:35–10:00）

  1. 从“小市值观察”中选3–5只；
  2. 限价单买入；
  3. 记录成本、买入日期。

第四步：持仓监控

  • 每日收盘前检查：是否破10日线？是否盈利≥8%？
  • 第5日14:50强制清仓。

第五步：每周调仓

  • 每周一重复选股，周二调仓，不每日操作。

⚠️ 风险提示

  • 小市值策略在大盘股主导行情中会持续跑输；
  • 严禁在熊市或小盘股暴跌期使用；
  • 单只股票成交额<2000万时，避免买入；
  • 建议先用模拟盘试运行1个月。

✅ 一句话总结：
放弃波动率计算和尾盘交易，聚焦“低价+小市值+非亏损+早盘买入+5日清仓”，借助问财筛选+人工排序，普通散户可稳健执行。
