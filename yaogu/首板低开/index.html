首板低开策略（散户实操优化版）

1. 策略分类和核心逻辑

  • 一级分类：方向性策略
  • 二级分类：事件驱动策略（核心博弈“首板涨停”后“低开”这一特定事件模式）
  • 核心逻辑：捕捉强势股（昨日首次涨停）在次日因情绪或洗盘低开3%-4%时的短线反弹机会。策略假设低开是短暂错杀，股价当日有望冲高，通过开盘买入、日内冲高卖出，获取1%-5%的短线收益。

2. 选股条件（适配普通软件）

  • 选股范围：A股主板（排除科创板688/689、创业板300开头、北交所4/8开头、ST/*ST、停牌股、上市不足250天新股）

  • 筛选方法（全部可通过同花顺/通达信/问财实现）：

  • 基础过滤：

    • 非ST、非科创板、非创业板、非北交所、非新股（上市>250天）
  • 昨日涨停：收盘价 = 涨停价（涨幅≈10%，主板）
  • 首板确认：昨日涨停，但前一日未涨停（即非连板）
  • 今日低开：开盘价在昨日收盘价的96%–97%之间（即低开3%–4%）

  • 简化位置要求（可选）：60日内最高价 > 当前价 × 1.3（即当前价处于60日相对低位，避免高位首板）

  • 更新频率：每日盘前（9:15–9:25）

✅ 问财语句示例（同花顺问财）：
昨日涨停，非ST，主板，上市天数>250，昨日非连续涨停，今日开盘价<昨日收盘价*0.97，今日开盘价>昨日收盘价*0.96
或更精确版：
涨停家数中，非ST，代码不以300、688、689、4、8开头，上市超过250天，昨日涨停且前日未涨停，今日开盘跌幅介于3%到4%

✅ 通达信选股公式（可保存为条件选股）：
```pascal { 首板低开策略 - 散户版 } ZT := C/REF(C,1) >= 1.095 AND C = H; { 昨日涨停 } NOT_ZT_YESTER := REF(C,2)/REF(C,3) < 1.095; { 前日未涨停，确保首板 } LOW_OPEN := O/REF(C,1) >= 0.96 AND O/REF(C,1) <= 0.97; { 今低开3%-4% } NOT_NEW := FINANCE(42) > 250; { 上市天数>250 } NOT_ST := NAMELIKE('ST')=0 AND NAMELIKE('*ST')=0; MAIN_BOARD := CODELIKE('60') OR CODELIKE('00');

ZT AND NOT_ZT_YESTER AND LOW_OPEN AND NOT_NEW AND NOT_ST AND MAIN_BOARD; ```

3. 买入条件

  • 触发信号：集合竞价结束（9:25）后，股票满足上述全部条件
  • 确认方式：
  • 9:25后，在问财或自选股中查看符合条件的股票列表
  • 人工核对：是否真为“首板”（看K线：昨日涨停，前日未涨停）
  • 执行时点：9:30 开盘后立即买入
  • 委托建议：
  • 使用限价单，价格设为开盘价或当前买一价
  • 若开盘后快速拉升，可放弃该股（避免追高）

4. 卖出条件（简化+明确）

  • 上午止盈（11:28）：
  • 若当前价 > 买入成本 且 未封涨停 → 全部卖出
  • 下午清仓（14:50）：
  • 无论盈亏，只要未封死涨停（即涨停板有打开或未触及）→ 全部卖出
  • 例外情况：
  • 若股票在14:50前已封死涨停且封单稳定，可持有至收盘（但散户建议仍卖出，避免尾盘炸板）

📌 操作提示：设置手机闹钟提醒 11:28 和 14:50，避免遗忘。

5. 仓位管理

  • 资金分配：若选出 N 只股票，则每只分配 总可用资金的 1/N
  • 例：总资金10万，选出2只 → 每只买5万
  • 最大持股数限制：建议 不超过3只（避免过度分散、操作困难）
  • 无加仓：单日单股仅买入一次
  • 空仓规则：若无符合条件股票，当日空仓；前一日持仓必在当日清空

6. 风控管理

  • 个股风控：
  • 低开3%-4%天然限制初始亏损（最大浮亏约4%）
  • 排除高位首板（通过60日相对低位辅助判断）
  • 日内交易，杜绝隔夜黑天鹅
  • 组合风控：
  • 最多3只股票，自动分散
  • 单日总亏损超5%时，次日暂停交易（可选纪律）
  • 系统风控：
  • 软件自动过滤ST、新股、非主板，降低踩雷概率

7. 手动实操步骤（纯人工+普通软件版）

第一步：盘前准备（9:15–9:25）

  1. 打开同花顺问财（网页或APP）
  2. 输入问财语句（见上文），获取初步股票池
  3. 人工复核每只股票：
  4. 看K线：确认昨日是第一个涨停板（前日未涨停）
  5. 看开盘价：确认低开在3%-4%之间（问财可能有延迟，以9:25集合竞价为准）
  6. 剔除明显问题股（如业绩暴雷、重大利空新闻）

第二步：开盘决策（9:25–9:30）

  1. 9:25集合竞价结束，刷新问财或自选股
  2. 最终确定买入名单（通常0–3只）
  3. 计算每只买入金额（总资金 ÷ 股票数）
  4. 准备好交易界面，挂单价格设为开盘价

第三步：盘中执行

  • 9:30：立即下单买入（限价单，快速成交）
  • 11:28：检查持仓，符合条件则卖出
  • 14:50：清仓所有未涨停持仓

第四步：纪律与记录

  • 必须空仓收盘
  • 每日记录：股票代码、买入价、卖出价、盈亏、是否符合逻辑
  • 每周复盘：统计胜率、平均收益、失败原因（如“低开低走”“冲高回落”）

⚠️ 风险提示（散户特别注意）

  • 该策略非稳赚，在市场退潮期（如连续跌停潮）胜率会显著下降
  • 手动操作存在延迟、漏单、情绪干扰风险
  • 建议先用模拟盘或≤10%资金试运行1个月，验证个人执行力
  • 若连续3日亏损，暂停策略，检查市场环境是否适合

✅ 一句话总结：
“首板+低开3%-4%+主板+非连板”是核心，纪律执行是盈利关键。借助问财/通达信可实现筛选，但需人工复核+严格守时。
