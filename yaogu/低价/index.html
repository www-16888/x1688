低价捕手策略

1. 策略分类和核心逻辑

  • 一级分类：方向性策略
  • 二级分类：均值回归与趋势跟踪结合
  • 核心逻辑：通过筛选低价、低成交但活跃度高的股票，捕捉短期波动机会，利用严格的买卖条件控制风险，在资金安全的前提下追求稳定收益。

2. 选股条件

  • 选股范围：全市场A股
  • 筛选方法：
  • 价格筛选：股价 < 10元
  • 流动性筛选：成交额 < 5000万元
  • 活跃度筛选：振幅 > 1%
  • 更新频率：每日

3. 买入条件

  • 触发信号：同时满足所有选股条件
  • 排序规则：当出现多只符合条件的个股时，选择换手率最小的股票
  • 执行时点：满足条件当日买入

4. 卖出条件

  • 技术卖出：出现KDJ死叉信号立即卖出
  • 时间卖出：持股周期达到6个交易日强制卖出
  • 执行原则：满足任一卖出条件即执行卖出操作

5. 仓位管理

  • 个股仓位：单只股票持仓比例根据总体资金管理规则确定
  • 持股周期：最长持有6个交易日
  • 换仓频率：根据选股结果和卖出信号灵活调整

6. 风控管理

  • 价格风控：通过低价股选择控制单笔投资金额
  • 流动性风控：选择成交额适中的股票避免流动性风险
  • 技术风控：KDJ死叉作为及时止损信号
  • 时间风控：6天持有期限避免资金长期占用

7. 手动实操步骤（纯人工交易版）

第一步：盘后筛选（每个交易日收盘后）

  1. 准备工具：
  2. 股票交易软件
  3. 股票筛选器

  4. 技术指标分析工具

  5. 基础筛选：

  6. 设置筛选条件：股价 < 10元
  7. 添加条件：昨日成交额 < 5000万元

  8. 添加条件：昨日振幅 > 1%

  9. 建立候选池：将符合基础条件的股票加入观察列表

第二步：深度分析（次日开盘前）

  1. 换手率排序：
  2. 计算候选股票的最新换手率
  3. 按换手率从小到大排序

  4. 选择换手率最小的1-2只股票

  5. 最终确认：确认选中的股票技术形态良好

第三步：买入执行（开盘后）

  1. 买入操作：
  2. 开盘后观察选中股票的表现
  3. 选择合适的时机买入

  4. 记录买入价格和日期

  5. 仓位控制：根据总体资金管理规则确定买入金额

第四步：持仓监控

  1. 技术指标监控：
  2. 每日检查持仓股的KDJ指标

  3. 关注是否出现死叉信号

  4. 时间监控：

  5. 记录每只股票的买入日期
  6. 计算已持有天数

第五步：卖出执行

  1. 技术卖出：
  2. 一旦出现KDJ死叉，立即卖出

  3. 不犹豫，严格执行

  4. 时间卖出：

  5. 持有满6个交易日，无论盈亏均卖出
  6. 为新一轮选股腾出资金

第六步：纪律维护

  1. 每日复盘：
  2. 检查持仓情况
  3. 更新持有天数计数

  4. 准备下一轮选股

  5. 严格执行：

  6. 坚决按KDJ死叉信号卖出
  7. 坚决按6天期限换股
  8. 保持策略的一致性

风险提示：该策略主要针对低价小盘股，波动性相对较大。KDJ指标在震荡市中可能出现频繁交叉，需要结合其他指标确认。严格执行持有期限是控制风险的关键。
