布林带通道突破趋势策略（散户实操版 · 同花顺/通达信友好）

1. 策略分类与核心逻辑

  • 一级分类：方向性策略
  • 二级分类：趋势跟踪 + 波动扩张结合
  • 核心逻辑：
    利用布林带宽度变化捕捉趋势启动点，当布林带收缩后突然扩张且价格突破上轨时，通常意味着趋势开始形成。本策略追求顺势介入、稳健持有、严格止损的趋势型收益。

2. 选股条件（完全适配问财语句）

✅ 选股范围

  • 全市场A股（主板为主）

❌ 排除股票（同花顺“条件选股”中勾选）

  • ST/*ST/退市股
  • 新股（上市未满250天）
  • 流通市值 < 50亿元（F10可查）
  • 当日涨停或跌停股（盘后自动过滤）

🔍 技术信号（问财语句实现）

非ST，非科创板，非创业板，主板，流通市值大于50亿，布林带宽度较前一周缩小至少30%，今日收盘价突破布林带上轨，成交量大于20日均量的1.5倍，MACD红柱持续增长，所属行业近一月涨幅排名前30%

✅ 说明： - 布林带宽度较前一周缩小至少30%：表示近期震荡收敛； - 今日收盘价突破布林带上轨：确认突破； - 成交量大于20日均量的1.5倍：放量突破； - MACD红柱持续增长：趋势确认； - 所属行业近一月涨幅排名前30%：提升成功率的关键过滤。

3. 买入条件（T+1合规）

  • 买入时机：次日尾盘 14:45–14:55（避免早盘追高）
  • 买入前提：
  • T日满足上述技术信号
  • T+1日股价仍在布林带上轨上方
  • T+1日成交量未显著萎缩
  • 仓位控制：单只 ≤ 20% 总资金
  • 最大持仓：同时持有 ≤ 3只
  • 买入方式：市价单（尾盘流动性好）

⚠️ 关键改进：放弃“9:35追突破”，改为次日尾盘低吸，大幅提升胜率。

4. 卖出条件（简单明确）

情况 操作
止盈 持有期间累计涨幅 ≥ 18%，次日冲高卖出
止损 收盘价跌破布林中轨，次日开盘卖出
时间止损 持有满20个交易日仍未达18%，第21日开盘清仓
波动风控 若布林带带宽增速 > 100%，暂停开新仓

✅ 不再依赖复杂加减仓规则，改用布林中轨破位这一清晰信号。

5. 仓位与风控（极简但有效）

项目 规则
单票仓位 ≤ 20%
最大持仓 3只
空仓条件 上证指数 < MA20 或 市场连板高度 ≤ 2板
流动性要求 日成交额 ≥ 3亿元（F10可见）
组合风控 账户总回撤 ≥ 5%，暂停开仓1周

6. 手动实操步骤（每日10分钟）

🕒 第一步：盘后选股（15:30–16:00）

  1. 打开 同花顺 → i问财
  2. 输入以下语句并点击“选股”： text 非ST，非科创板，非创业板，主板，流通市值大于50亿，布林带宽度较前一周缩小至少30%，今日收盘价突破布林带上轨，成交量大于20日均量的1.5倍，MACD红柱持续增长，所属行业近一月涨幅排名前30%
  3. 将符合的加入自选股（最多5只）

🕒 第二步：次日尾盘决策（14:45–14:55）

  1. 查看自选股：
  2. 是否仍在布林带上轨上方？
  3. 成交量是否未显著萎缩？
  4. 所属板块是否有持续热度？
  5. 选1–2只最强标的，市价买入 ≤20%仓位

🕒 第三步：持有与卖出（T+2日起）

  • 每日收盘检查：是否破布林中轨？是否达18%？是否满20天？
  • 按第4节规则执行

🕒 第四步：每周复盘（周日晚）

  • 统计：胜率、平均持有天数、最大回撤
  • 若胜率 < 40%，暂停并反思

7. 同花顺操作指南（零公式）

功能 路径
调出布林带 K线图按 BOLL 键 → 显示布林带上下轨及带宽
查看布林带宽度 在K线图右键菜单中选择“布林带参数设置”查看当前带宽
查看MACD K线图按 MACD 键 → 显示MACD指标
查看行业涨幅 首页 → 行情 → 板块排行 → 近一月涨幅排序
查看流通市值 F10 → 基本资料

8. 策略优势

  • 完全T+1合规：买入次日可卖，无制度冲突
  • 零计算、零编程：全靠肉眼+软件默认指标
  • 避开假突破高峰：尾盘低吸，而非早盘追高
  • 信号清晰：布林带收缩后扩张+放量突破，双重确认
  • 适合上班族：每日仅需10分钟

9. 风险提示

  • 仅在大盘上升趋势中使用（上证站上MA20）；
  • 绝不追高：高开>5%直接放弃；
  • 严格止损：破布林中轨无条件离场；
  • 震荡市空仓：若市场无主线，宁可错过。
