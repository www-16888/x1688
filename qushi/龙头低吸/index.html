龙头股低吸策略（散户手动简化版）

1. 策略分类和核心逻辑

一级分类：技术面+情绪面策略（简化执行版）
二级分类：龙头股低吸策略（非打板版）
核心逻辑：基于市场情绪周期理论，识别近期市场中连板数最多或板块地位最强的龙头股，在其首次明显分歧回调（如炸板、放量滞涨、回踩均线）后，于次日或尾盘低吸介入，避免追高打板，通过合理止盈止损，捕捉龙头第二波上涨机会，追求稳健的短线超额收益。

2. 选股条件

选股范围：全市场A股（主板为主）

排除范围：
- 科创板、北交所股票
- ST股及其他退市标签股
- 上市未满60天的次新股
- 当日停牌或涨跌停股票

核心筛选标准：
- 龙头定义：近5个交易日内出现过2连板及以上，或为当前市场最高连板股/板块核心标的（可通过同花顺“涨停棱镜”或“连板天梯”查看）
- 数量限制：最多同时观察3只龙头股，最终持仓不超过2只

技术面筛选：
- 回调信号：龙头股出现以下任一情形：
- 涨停被打开（炸板）且收盘未回封；
- 连续上涨后首次放巨量（成交量 > 前5日均量2倍）但涨幅 < 3%；
- 股价回调至10日均线附近（±2%）且未破20日均线。
- 黑名单机制：若某龙头股买入后亏损超5%，3日内不再参与。

3. 买入条件

交易频率：低频择机（平均每月1–3次，非每日交易）

买入规则：
- 仅当龙头股出现明确分歧回调信号后才考虑买入；
- 不参与开盘打板，放弃“排队买涨停”行为；
- 买入时机：尾盘14:45–14:55（避免盘中诱多）；
- 买入前提：当日涨幅 ≤ 3%，且大盘未大跌（上证跌幅 < -1%）；
- 资金分配：单只股票仓位 ≤ 25%，总股票仓位 ≤ 50%。

买入方式：
- 手动市价单买入（尾盘流动性充足，无需限价排队）；
- 不追求买在最低点，只求买在“安全区域”。

4. 卖出条件

止损卖出：
- 从买入成本起算，亏损 ≥ 7% 立即卖出；
- 若买入后次日低开低走，无反抽意愿，可提前止损。

条件卖出（14:50前）：
- 持有满3个交易日仍未达止盈目标，第4日开盘卖出；
- 股价再次涨停，次日冲高（如+3%以上）卖出；
- 板块整体退潮（同板块多只股大跌），无论盈亏减仓或清仓。

特殊情况：
- 若买入后当日涨停，可持有至次日冲高卖出；
- 遇重大利好（如政策支持、业绩超预期），可延长持有至5日。

5. 仓位管理

持仓数量：最多2只股票（避免精力分散）

资金分配：
- 单只股票初始仓位10%～25%；
- 总股票仓位不超过50%，其余资金持有货币ETF（511880）。

调仓频率：不每日调仓，仅在出现新龙头分歧机会时替换；

风险控制：
- 不满仓、不杠杆、不All in单一个股；
- 同一板块最多持有1只龙头。

6. 风控管理

盘前风控：
- 不参与无明确逻辑的“跟风涨停”；
- 只做市场公认的主线龙头（如AI、新能源、消费等当前热点）。

盘中风控：
- 不因盘中拉升临时追高；
- 不因小幅浮亏补仓摊薄。

流动性风控：
- 仅参与日成交额 ≥ 3亿元的龙头股，确保进出顺畅。

情绪风控：
- 若市场最高连板数 ≤ 2板，视为“情绪退潮期”，暂停新开仓；
- 上证指数跌破250日均线时，仓位上限降至20%。

纪律风控：
- 7%止损线不可突破；
- 不持仓过周末（除非已涨停或重大利好）；
- 每月最多交易3次，宁可错过，不可做错。

7. 情绪周期监控（简化版）

情绪指标：
- 观察同花顺“连板天梯”或东方财富“涨停板”栏目中的最高连板数；

周期判断：
- 最高连板 ≥ 4板，且梯队完整（3板、2板有跟风）→ 上升期，可参与；
- 最高连板 ≤ 2板，或龙头连续A杀 → 退潮期，空仓观望。

情绪记录：
- 每周日晚简单记录：“本周龙头是谁？是否赚钱效应？”
- 不需复杂模型，凭常识判断即可。

8. 手动实操步骤（纯人工交易版）

第一步：盘后观察（每日16:00–17:00）
- 打开同花顺“涨停棱镜”或“连板天梯”；
- 标记当前市场最高连板股及板块核心标的；
- 加入自选股，观察是否出现分歧信号。

第二步：分歧识别（次日起）
- 观察龙头是否炸板、放巨量、回踩均线；
- 确认板块是否仍有支撑（至少1只跟风股强势）。

第三步：买入执行（尾盘14:45–14:55）
- 确认当日涨幅 ≤ 3%；
- 确认大盘稳定；
- 手动下单，设置7%止损条件单。

第四步：持仓管理（买入后）
- 第2日：若冲高+5%以上，可部分止盈；
- 第3–4日：未达目标则评估趋势，第4日离场；
- 任何时点破7%止损，立即卖出。

第五步：收盘总结（每日15:30后）
- 记录交易理由与结果；
- 更新“近期失败龙头”黑名单；
- 判断市场情绪是否进入退潮。

第六步：纪律维护
- 不因“怕卖飞”而放弃止损；
- 不因“市场热闹”而强行开仓；
- 只做高确定性机会，其余时间空仓学习。

策略理论基础（简化版）

  • 龙头效应：市场最强股在分歧后常有反包机会；
  • 情绪周期：退潮期空仓，上升期轻仓试错；
  • 低吸优于打板：回避开盘抢筹劣势，提升盈亏比；
  • 强者恒强：真龙头回调后往往率先修复。

交易执行要点

  • 耐心等待：每月可能只有1–2次机会；
  • 尾盘介入：避开早盘情绪干扰；
  • 严格止损：7%是生命线；
  • 轻仓试错：单票≤25%，总仓≤50%。

风险提示

  • 龙头股波动大，单日回撤可达5%～8%；
  • 情绪退潮时，龙头也可能连续大跌；
  • 该策略依赖对“主线”和“龙头”的正确识别，判断错误将导致亏损；
  • 最大风险仍是纪律执行——必须接受“小亏离场”的设定。
