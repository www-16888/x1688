价值成长RSI择时策略（散户实操优化版 · A股合规版）

1. 策略分类和核心逻辑

  • 一级分类：基本面+技术面复合策略
  • 二级分类：价值成长RSI择时策略
  • 核心逻辑：结合同花顺问财自然语言一键筛选高ROE、高增长、合理估值的小市值股票，并通过RSI技术指标在月度调仓时择时买入与止盈卖出。策略简化了财务数据获取、降低了调仓频率、避免了过度交易，全部条件均可在免费软件中实现，符合A股T+1、100股整数倍、交易时段等规则，适合普通散户每月操作20分钟。

2. 选股条件

  • 选股范围：全市场A股（主板+中小板）
  • 排除范围：
  • 科创板（688/689）、北交所（4/8开头）
  • ST、*ST、退市风险股
  • 上市不满375天的新股

  • 停牌股

  • 基本面筛选标准：

  • 估值指标：
    • 市净率（PB）：0–5倍（原10倍过高，降低估值要求）
    • 市现率（PCF）：< 5倍
  • 盈利能力：
    • 净资产收益率（ROE）：> 15%（原30%过高，调整为更现实的标准）
    • 销售毛利率：> 20%（原30%过高，调整为更现实的标准）
  • 成长能力：
    • 净利润同比增长率：> 50%（原150%过高，调整为更现实的标准）

  • 市值偏好：

    • 按总市值从小到大排序
    • 初步筛选50只最小市值股票

  • 技术面筛选标准：

  • RSI指标：
    • 14日RSI < 50（处于弱势区间）
    • 选择被低估的优质股票
  • 最终持仓：选择前10只股票构建组合

✅ 同花顺问财语句（每月第一个交易日9:00直接复制）：
主板+中小板股票，非ST，上市天数大于375，PB小于5，PCF小于5，ROE大于15，销售毛利率大于20，净利润同比增长大于50，按总市值从小到大排序

3. 核心筛选流程

第一步：月度选股（每月第一个交易日9:00–9:20）

  1. 基础筛选：
  2. 使用上述问财语句筛选符合条件的股票；
  3. 按总市值从小到大排序；
  4. 排除停牌股票。
  5. 技术面筛选：
  6. 在通达信或同花顺中计算每只股票的14日RSI值；
  7. 选择RSI < 50的股票（处于弱势但基本面优质）；
  8. 取前10只构建最终投资组合；
  9. 记录买入名单（代码、名称、市值、RSI值）。

第二步：调仓准备（周一开盘前）

  1. 持仓分析：
  2. 检查当前持仓情况；
  3. 识别不在新选股名单中的持仓；
  4. 准备调仓对比名单。
  5. 资金规划：
  6. 计算可用资金总额；
  7. 确定每只新股的买入金额（等分资金）。

4. 买入条件

  • 调仓频率：每月一次（每月第一个交易日）
  • 持仓数量：固定持有10只股票
  • 资金分配：
  • 可用现金等分买入每只股票；
  • 每只买入数量为100股整数倍。
  • 执行时间：9:35–10:00（避开集合竞价，确保成交）

5. 卖出条件

  • 定期调仓卖出：
  • 每月调仓日，卖出不在新选股名单中的持仓股；
  • 保留仍在名单中的股票（减少交易频率）。
  • 技术止盈：
  • 每日检查持仓股的14日RSI值；
  • 若RSI > 70，立即卖出该股票；
  • 锁定盈利，避免回调。
  • 卖出时点：
  • 次日开盘卖出（若RSI超买）；
  • 其他情况下，次日开盘卖出。

6. 仓位管理

  • 持仓数量：固定持有10只股票
  • 资金分配：等权重，每只约占10%资金
  • 总仓位：
  • 若选出≥10只 → 满仓运作；
  • 若选出<10只 → 剩余资金买入货币ETF（511880）。
  • 行业分散：强制覆盖≥3个不同行业（消费、制造、医药等）

7. 风控管理

  • 基本面风控：严格的财务指标筛选确保标的质量；
  • 估值风控：控制估值水平，避免过高溢价；
  • 技术风控：RSI指标及时止盈，控制回撤；
  • 分散风控：10只股票分散投资，降低个股风险；
  • 流动性风控：排除高风险板块股票；
  • 止损风控：收盘价连续2日低于20日均线 → 次日开盘卖出。

8. 手动实操步骤

第一步：盘前准备（每个交易日9:05）

  1. 持仓管理：
  2. 更新当前持仓列表；
  3. 准备当日监控工作。

第二步：月度选股（每月第1个交易日9:00–9:20）

  1. 基础筛选：
  2. 获取全市场股票列表；
  3. 排除科创、北交、ST、次新、停牌股票；
  4. 筛选PB在0–5倍、PCF<5倍的股票；
  5. 筛选ROE>15%、毛利率>20%的股票；
  6. 筛选净利润增长>50%的股票；
  7. 按市值排序选择前50只最小市值股票。
  8. 技术面筛选：
  9. 计算50只候选股的14日RSI值；
  10. 选择RSI < 50的股票（处于弱势但基本面优质）；
  11. 取前10只构建最终投资组合。

第三步：月度调仓（每月第1个交易日9:30）

  1. 调仓执行：
  2. 卖出所有不在新组合中的持仓；
  3. 等分资金买入10只目标股票；
  4. 完成月度调仓。

第四步：每日监控（每个交易日9:30）

  1. 技术止盈：
  2. 计算每只持仓股的14日RSI值；
  3. 如RSI > 70，立即卖出该股票；
  4. 实现及时止盈，锁定收益。

第五步：持续优化

  1. 参数回顾：
  2. 回顾基本面筛选标准的有效性；
  3. 优化RSI参数的敏感性；
  4. 调整持仓数量与资金分配。
