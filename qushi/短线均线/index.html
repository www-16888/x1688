短线均线突破策略（散户实操版 · 同花顺/通达信友好）

1. 策略分类与核心逻辑

  • 一级分类：技术面短线策略
  • 二级分类：均线系统 + 量价配合（T+1低频版）
  • 核心逻辑：
    利用5日与10日均线死叉后快速金叉、MACD低位金叉、放量回升三大信号，捕捉短期调整结束后的反弹机会。
    买入后持有1–3天，不日内交易，次日可卖。

2. 选股范围与排除规则（同花顺操作）

✅ 选股范围

  • 全市场A股（主板为主）

❌ 排除股票（在同花顺“选股器”中勾选排除）

  • 创业板（300开头）
  • 科创板（688开头）
  • 北交所（8开头）
  • ST/*ST/退市股
  • 上市未满250天的次新股
  • 当日涨跌停或停牌股（盘后选股时自动过滤）

📌 操作路径（同花顺）：
首页 → 选股 → 条件选股 → 基础条件 → 勾选“非ST”、“非创业板”、“非科创板”、“上市满250天”

3. 技术信号（全部使用软件内置指标）

信号1：均线系统调整（使用MA指标）

  • 打开个股K线图，调出 MA5、MA10、MA20、MA30（默认参数即可）
  • 昨日：MA5 下穿 MA10（死叉）
  • 今日：MA5 上穿 MA10（金叉）
  • 同时：MA20 和 MA30 今日向上（趋势未破）

✅ 判断方法：肉眼观察K线图，无需计算。死叉/金叉看均线交叉点。

信号2：MACD低位金叉（使用MACD指标）

  • 调出MACD指标（默认参数：12,26,9）
  • DIF 与 DEA 均在0轴上方
  • MACD柱由绿转红（或红柱开始放大）
  • 金叉发生在最近1–2个交易日

✅ 判断方法：只要看到MACD在0轴上、红柱初现，即视为有效。

信号3：量能配合（使用VOL指标）

  • 观察成交量柱：昨日缩量下跌，今日放量上涨
  • 今日成交量 > 前5日平均成交量（肉眼判断：今日量柱明显高于前几天）

✅ 无需计算均线，只要“今天比前几天明显放量”即可。

信号4：价格形态

  • 前日下跌，今日上涨（形成“阳包阴”或止跌阳线）
  • 今日收盘价 > 昨日最高价（强势反包，加分项，非必须）

4. 买入条件（T+1合规）

  • 买入时机：尾盘14:45–14:55（避免盘中诱多）
  • 买入前提：
  • 今日已出现 MA5上穿MA10
  • MACD红柱初现
  • 成交量明显放大
  • 股价未涨停（确保能买入）
  • 仓位控制：单只股票 ≤ 20% 总资金
  • 持仓数量：最多同时持有2只（避免精力分散）
  • 买入方式：市价单（尾盘流动性好，无需限价）

⚠️ 注意：今日买入，最早明日9:30才能卖出（T+1规则）

5. 卖出条件（次日或持有期内）

✅ 止盈卖出（满足任一即卖）：

  • 持有第2日：冲高+5%以上，可分批止盈
  • 持有第3日：无论盈亏，收盘前清仓（避免过夜不确定性）
  • 股价再次涨停：次日冲高卖出（不板上持股）

✅ 止损卖出（硬性纪律）：

  • 从买入成本起算，亏损 ≥ 6%，次日开盘立即卖出
  • 若次日低开低走、无反抽，可在10:00前止损

✅ 技术破位卖出：

  • 次日收盘跌破MA10，无论盈亏，第3日开盘卖出

6. 仓位与风控（散户友好）

项目 规则
单票仓位 ≤ 20%
最大持仓 2只股票
空仓条件 大盘5日均线下行 或 最高连板 ≤ 2板
流动性要求 日成交额 ≥ 1亿元（同花顺K线图可见）
情绪风控 市场连续3日超3000股下跌 → 暂停开仓

7. 手动实操步骤（每日10分钟完成）

🕒 第一步：盘后选股（15:30–16:00）

  1. 打开同花顺 → 选股 → 条件选股
  2. 设置基础过滤（非ST、非创、非科、上市满250天）
  3. 手动浏览“涨幅3%~7%”板块，找符合以下特征的个股：
  4. 今日阳线，昨日阴线
  5. MA5上穿MA10（看K线图）
  6. MACD红柱初现（0轴上）
  7. 量比 > 1.2（同花顺F10中可见“量比”）
  8. 将候选股加入自选股（最多5只）

🕒 第二步：次日尾盘决策（14:45–14:55）

  1. 查看自选股中哪些仍符合：
  2. 未涨停
  3. 股价在MA10上方
  4. 板块未退潮
  5. 选择1只最强标的，市价买入 ≤20%仓位

🕒 第三步：次日卖出（T+1日）

  • 按第5节规则执行止盈/止损
  • 卖出后更新自选股

🕒 第四步：每周复盘（周日晚）

  • 回顾本周交易：胜率、平均盈亏比
  • 若连续2周亏损，暂停1周

8. 同花顺/通达信指标调用指南

指标 调用方式
均线（MA） K线图界面按 MA 键，或右键 → 主图指标 → MA
MACD 按 MACD 键，或副图指标 → MACD
成交量（VOL） 默认显示在K线下方
量比 F10 → 基本资料 → 实时数据（或分时图上方）
所属板块/涨停家数 首页 → 涨停聚焦 / 行业板块

✅ 无需编写公式，无需计算，全部肉眼观察即可

9. 策略优势

  • 完全T+1合规：买入次日可卖，无制度冲突
  • 零计算负担：依赖软件现成指标，肉眼判断
  • 信号明确：金叉+放量+MACD红柱，三者共振
  • 风险可控：6%止损 + 3日持有上限 + 20%仓位
  • 适合上班族：每日仅需10–15分钟操作

10. 风险提示

  • 该策略在震荡市或反弹初期效果最佳，单边下跌市慎用；
  • 均线金叉可能是“假信号”，必须配合量能与MACD验证；
  • 绝不全仓、绝不追高、绝不持仓过周末（除非涨停）；
  • 若大盘破位（如上证跌破20日线），应主动空仓。
