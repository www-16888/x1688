低PB小市值低换手策略

1. 选股

✅ 选股范围：

  • 主板A股（60/00开头），排除科创板（688）、创业板（300）
  • 非ST、*ST股票（财务状况异常）
  • 正常交易状态（非停牌、非涨停）

✅ 核心指标：

条件 标准 查看位置
低市净率（PB） PB < 1.5倍 同花顺F10 → “主要财务指标”或行情界面
小市值 总市值 < 100亿元 F10 → “公司概况” 或 行情界面
低换手率 近5日平均换手率 < 1% 行情界面或同花顺“个股资料”中的“换手率”

💡 为什么选这些阈值？
- PB < 1.5倍：通常认为低于1.5倍的股票具有一定的估值安全边际。
- 市值 < 100亿：避免过大市值导致成长性不足，同时控制流动性风险。
- 换手率 < 1%：表明市场关注度较低，可能被低估。

2. 买入条件

✅ 买入规则：

  • 调仓时间：每月第一个交易日 9:30
  • 持仓数量：固定 10只股票
  • 买入方式：
  • 等权重分配（总资金 ÷ 10）
  • 优先选择行业分散的股票（避免全买银行或全买地产）
  • 若某只股票涨停无法买入，则跳过，从备选中补足10只

3. 卖出条件

✅ 卖出规则：

  • 常规卖出：调仓日卖出不在新名单中的股票
  • 强制卖出：不符合新选股标准的持仓全部卖出

4. 仓位管理

✅ 仓位控制：

  • 总持仓：通常满仓（除非大盘极度高估）
  • 单个股票：10%（严格等权）
  • 调仓频率：每月1次（降低摩擦成本）
  • 行业分散：尽量覆盖3个以上行业（如银行、电力、交运、化工、公用事业等）

5. 风控管理

✅ 风险控制：

  • 质量风控：排除ST、*ST股票，确保财务健康
  • 流动性风控：排除停牌股票，确保可正常交易
  • 价格风控：排除涨停股票，确保可买入
  • 情绪风控：不追热点、不炒题材、不碰“看不懂”的公司

6. 手动实操步骤（纯人工版）

📅 每月第一个交易日操作流程（9:00–9:30）

第一步：用问财快速初筛（9:00–9:10）

在 同花顺i问财（https://www.iwencai.com）输入：

主板，非ST，PB<1.5，市值<100亿，近5日平均换手率<1%，上市满1年

✅ 结果通常为50~100只股票，导出或截图。

第二步：人工精筛（9:10–9:25）

对问财结果，逐个检查F10： 1. 是否连续盈利？（近3年至少有盈利）
2. 是否有大额负债或亏损？（看“主要财务指标”）
3. 所属行业是否稳健？（优先选：银行、电力、高速、港口、煤炭、家电等）
4. 当日是否可交易？（非涨跌停、非停牌）

选出10只，按“行业分散 + PB最低 + 换手率最低”排序。

第三步：调仓执行（9:30）

  1. 卖出：不在新名单的持仓（涨停股除外）
  2. 买入：等分资金买入新10只
  3. 记录：在Excel登记股票、成本、行业、PB、换手率

7. 通达信 / 同花顺友好支持

✅ 问财选股语句（直接复制）：

主板，非ST，PB<1.5，市值<100亿，近5日平均换手率<1%，上市满1年
