基本面技术面共振策略

1. 策略分类和核心逻辑

  • 一级分类：方向性策略
  • 二级分类：趋势跟踪与基本面量化结合
  • 核心逻辑：通过基本面筛选排除高风险标的，结合技术面指标捕捉趋势启动信号，在严格控制仓位和风险的前提下，追求高盈亏比的短线交易机会。

2. 选股条件

  • 选股范围：全市场A股
  • 筛选方法：
  • 基本面过滤：
    • 非ST股票
    • 非次新股、非新股（通常指上市超过1年）
    • 市净率（PB）≤ 5
  • 技术面信号：
    • RSI指标日线拐头向上
    • BOLL布林带开口张开
    • K线形态出现长上影线
  • 更新频率：每日开盘前

3. 买入条件

  • 触发信号：同时满足基本面和技术面所有条件
  • 确认方式：在早上9:31，确认技术指标信号有效
  • 执行时点：上午9:31开盘后立即执行
  • 排序规则：按成交额由小到大排序，优先选择成交额较小的标的

4. 卖出条件

  • 止盈条件：涨幅超过13%立即止盈
  • 止损条件：跌幅达到6%立即止损
  • 强制卖出：持有满9个交易日，无论盈亏均卖出换股

5. 仓位管理

  • 个股仓位：单只股票最大持仓不超过总资金的20%
  • 持股数量：账户同时最大持有5只股票
  • 每日买入：单日最多买入2只新股票
  • 加减仓规则：不进行加仓操作，达到条件即卖出

6. 风控管理

  • 个股风控：严格的6%止损线和13%止盈线
  • 组合风控：通过持股数量限制和个股仓位控制实现分散投资
  • 时间风控：9天持有周期限制，避免资金长期占用
  • 流动性风控：优先选择成交额较小的股票，避免追高

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每个交易日前一晚或当日9:15前）

  1. 准备工具：
  2. 股票交易软件
  3. 技术指标分析工具（RSI、BOLL）

  4. 股票筛选器或Excel表格

  5. 基本面筛选：

  6. 排除所有ST股票
  7. 排除上市不满1年的新股和次新股

  8. 筛选市净率≤5的股票

  9. 建立观察池：将符合基本面条件的股票加入自选观察池

第二步：开盘分析（9:15-9:30）

  1. 技术指标确认：
  2. 检查观察池中股票的RSI指标是否拐头向上
  3. 确认BOLL布林带是否开口张开

  4. 观察前一日K线是否有长上影线形态

  5. 最终候选名单：记录同时满足所有技术条件的股票

第三步：开盘交易（9:31）

  1. 排序筛选：
  2. 查看候选股票的成交额
  3. 按成交额从小到大排序

  4. 选择前2只成交额最小的股票

  5. 买入操作：

  6. 计算单只股票买入金额：总资金 × 20%
  7. 使用限价委托方式买入
  8. 确保总持股数不超过5只

第四步：盘中监控

  1. 止损止盈监控：
  2. 设置价格预警：成本价上涨13%和下跌6%

  3. 达到预警立即执行卖出操作

  4. 持仓管理：

  5. 记录每只股票的买入日期
  6. 标记即将满9天的持仓

第五步：日常维护

  1. 持仓检查：
  2. 每日检查持仓是否达到9天持有期限

  3. 到期立即卖出，无论盈亏

  4. 数量控制：

  5. 确保总持股数不超过5只

  6. 每日新买入不超过2只

  7. 纪律执行：

  8. 严格按6%止损、13%止盈执行
  9. 坚决执行9天换股规则

风险提示：该策略为短线交易策略，交易频率较高，需要严格执行纪律。技术指标存在滞后性，可能遇到假信号。手动执行需要及时盯盘，避免错过买卖时机。
