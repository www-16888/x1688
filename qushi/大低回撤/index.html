大容量低回撤投资策略（散户简化版）

1. 策略的风险等级、分类和策略核心逻辑

  • 风险等级：R2–R3（稳健偏保守）
  • 策略分类：价值投资策略 / 基本面筛选策略
  • 策略核心逻辑：
    聚焦盈利稳定、估值合理、股价波动小的优质公司，避开高估值、高波动、业绩差的股票。
    通过半年调仓 + 持仓分散 + 止盈纪律，在控制回撤的前提下获取长期稳健收益。
    核心理念：买“便宜的好公司”，不追热点，不碰妖股。

2. 选股条件（人工可查、工具可筛）

  • 选股范围：沪深300成分股 或 中证红利指数成分股（质量高、流动性好、数据全）
  • 基础池：正常交易、非ST、非*ST、上市满1年以上
  • 筛选条件（四看原则）：
  • 看盈利：近3年净利润为正，且销售毛利率 > 20%（行业对比更佳）
  • 看估值：市盈率（PE_TTM）< 15，且市净率（PB）< 1.8
  • 看波动：近1年股价最大回撤 < 30%（对比沪深300）
  • 看分红：近3年有分红记录（体现现金流和股东回报意识）

✅ 无需计算：上述数据在同花顺F10、东方财富APP、雪球个股页均可直接查看。

3. 买入条件

  • 买入时机：每年6月第一个交易日 和 12月第一个交易日（半年调仓）
  • 买入条件：
  • 股票满足上述“四看”条件
  • 不在最近6个月内持仓过（避免反复买卖）
  • 当日非涨停、非跌停（避免追高或接飞刀）
  • 买入方式：
  • 等权重买入：总资金 ÷ 目标持股数（建议10~15只）
  • 按优先级买入：优先选分红稳定、行业龙头、毛利率更高的公司
  • 分批建仓（可选）：若大盘处于高位（如上证>3400），可分2天买入

4. 卖出条件

  • 定期调仓：半年调仓日卖出不符合新条件的股票
  • 止盈卖出：
  • 单只个股累计盈利 ≥ 30%，可考虑卖出一半，留一半博趋势
  • 若盈利回撤超50%（如从+40%跌到+20%），则全部止盈
  • 止损卖出：
  • 股价跌破买入价 -15%，无条件止损
  • 公司出现业绩暴雷、被ST、分红中断等基本面恶化信号，立即卖出
  • 不参与涨停博弈：若持仓股突然连续涨停（非基本面驱动），第二个涨停板打开时卖出

5. 仓位管理

  • 持仓数量：10~15只（兼顾分散与可操作性）
  • 仓位比例：
  • 单一个股：6%~10%（等权重）
  • 总仓位：
    • 大盘PE<12 或 沪指<3000 → 满仓
    • 大盘PE>18 或 沪指>3400 → 仓位≤70%，留现金
  • 动态调整：仅在调仓日调整，平时不频繁操作

6. 风控管理

  • 分散风控：10~15只跨行业（避免全买银行或全买消费）
  • 质量风控：只选有持续盈利和分红的公司
  • 估值风控：PE>20 或 PB>2 的股票一律不碰
  • 情绪风控：不追热点、不炒题材、不听消息
  • 止损纪律：-15%硬止损，保护本金第一

7. 手动实操步骤（纯人工交易版）

第一步：调仓日前准备（5月30日 / 11月30日）

  1. 打开同花顺/东方财富，进入“沪深300”或“中证红利”成分股列表
  2. 导出或截图成分股名单（约300只）

第二步：初筛（用问财/i问财快速过滤）

在 同花顺i问财（https://www.iwencai.com）输入以下语句（任选其一）：

✅ 问财选股语句（直接复制粘贴）：

沪深300成分股，非ST，上市满1年，市盈率<15，市净率<1.8，毛利率>20%，近3年净利润为正，近3年有分红

或

中证红利成分股，PE<15，PB<1.8，毛利率>20%，非ST，近一年最大回撤<30%

💡 结果通常为30~60只股票，人工再精筛。

第三步：人工精筛（肉眼观察F10）

对问财筛选出的股票，逐个打开F10资料，检查： - 是否近3年都分红？
- 是否有大额商誉或应收账款激增？（警惕财务风险）
- 所属行业是否处于下行周期？（如地产、教培等回避）
- 近1年股价是否平稳？（避免“过山车”个股）

最终选出10~15只，按“行业分散+龙头优先”原则确定。

第四步：调仓日操作（6月1日 / 12月1日 9:30）

  1. 卖出：不在新名单中的持仓股（涨停则顺延至开板日）
  2. 买入：等分资金买入新名单股票（避开近6个月持有过的）
  3. 记录：在Excel登记买入日期、成本、行业

第五步：日常监控（无需盯盘）

  • 每月查看一次持仓股F10，关注季度财报发布
  • 若个股触发止盈/止损条件，在收盘前5分钟手动操作
  • 其余时间不看盘、不操作

第六步：半年复盘

  • 统计组合收益 vs 沪深300
  • 检查哪些因子失效（如低PE股跑输）
  • 微调选股条件（如将PE上限从15调到18）

关键执行要点

  • ✅ 只用免费工具：同花顺、问财、F10足够
  • ✅ 不计算、不编程：所有数据肉眼可见
  • ✅ 低频操作：每年仅2次调仓，适合上班族
  • ✅ 严守纪律：不因“感觉要涨”而提前买入
  • ✅ 行业分散：金融、消费、制造、公用事业等至少覆盖3个行业
