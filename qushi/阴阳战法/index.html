三阳三阴战法策略（散户实操优化版 · A股合规版）

1. 策略分类和核心逻辑

  • 一级分类：技术面短线策略
  • 二级分类：K线形态 + 突破低吸
  • 核心逻辑：在符合A股T+1规则的前提下，识别近6个交易日出现“三连阳”后小幅回调（1–2根阴线） 的主板股票，于回调结束、再次放量上涨时低吸介入，持有1–3日博弈反弹。核心是“三阳蓄势 + 阴线洗盘 + 二次启动”，避免追高，专注可量化、可买入、可止损的形态。

2. 选股条件

  • 选股范围：A股主板（60/00开头）
  • 排除范围：
  • 创业板（300开头）、科创板（688/689开头）
  • ST、*ST、退市风险警示股票
  • 停牌股票
  • 上市不满200天的次新股（原1080天过严，调整为合理值）

3. 核心筛选流程

第一步：盘前形态初筛（使用问财自然语言）
筛选同时满足以下条件的股票：
1. 近6日内有连续3日阳线（收盘价 > 开盘价）
2. 最近1–2日为阴线回调（收盘价 < 开盘价）
3. 股价处于相对低位：当前价 < 近20日最高价 × 0.95（避免高位接盘）
4. 趋势未破：股价 > 20日均线
5. 流动性充足：昨日成交额 > 3000万元

✅ 同花顺问财语句（直接复制使用）：
主板股票，非ST，上市天数大于200，近6日有连续3天收盘价大于开盘价，最近2天有至少1天收盘价小于开盘价，当前价小于20日最高价的95%，股价高于20日均线，昨日成交额大于30000000

📌 说明：
- 问财支持“连续3天收盘价大于开盘价”等自然语言；
- “最近2天有阴线”确保回调发生；
- 结果通常为5–15只，人工复核K线形态即可。

第二步：人工K线复核（5分钟内完成）
打开问财结果中的个股K线图（日线），肉眼确认：
- 是否有清晰的“阳阳阳 → 阴/阴阴”结构（共4–6根K线）；
- 三阳期间温和放量，阴线缩量回调；
- 无长上影线或大阴线破位。
→ 剔除形态不标准的，保留3–5只。

4. 买入条件

  • 技术信号：
  • 通过问财筛选 + 人工K线确认；
  • 当日9:35后，股价站上昨日最高价（确认回调结束）；
  • 当前价未涨停（留有买入空间）。
  • 买入时机：9:35–10:00（连续竞价稳定后）
  • 仓位控制：
  • 最多持仓3只（不全仓单只）；
  • 等权重分配（如总资金6万，每只≤2万）；
  • 买入数量为100股整数倍（符合A股规则）。

5. 卖出条件

  • 止盈卖出：
  • 持仓第2日或第3日涨幅 ≥ 6% → 分批或全部卖出；
  • 涨停封板 → 可持有至次日冲高或开板卖出。
  • 止损卖出：
  • 买入后任一交易日收盘价 < 买入价 × 0.93（-7%硬止损）；
  • 盘中跌破20日均线且30分钟未收回 → 次日开盘卖出。
  • 强制清仓：
  • 最长持有3个交易日（T日买入 → T+3日14:55前清仓）。

✅ 完全符合T+1：T日买入 → T+1日起可卖出。

6. 仓位管理

  • 持仓数量：1–3只（新手建议1–2只）
  • 资金分配：等权重，单只 ≤ 总资金35%
  • 交易频率：有信号才交易，无信号空仓
  • 风险控制：单笔最大亏损 ≤ 7%，总仓位 ≤ 100%

7. 风控管理

  • 质量风控：排除创业板、科创板、ST、次新
  • 流动性风控：成交额 > 3000万，确保可成交
  • 技术风控：20日均线 + 缩量回调双重验证
  • 止损风控：-7%硬止损 + 均线破位离场
  • 分散风控：避免同行业集中（如不全买AI或新能源）

8. 交易频率

  • 选股计算：每个交易日9:00用问财完成
  • 买入操作：9:35–10:00（T日）
  • 卖出操作：T+1至T+3日 9:30–14:55
  • 清理操作：T+3日14:55前清仓，不留隔夜

9. 手动实操步骤

第一步：盘前筛选（9:00–9:20）
1. 问财粘贴语句，导出候选股；
2. 打开每只K线图，确认“三阳+1–2阴”形态；
3. 剔除高开>5%或低开<-3%的极端股；
4. 保留3只以内标的。

第二步：开盘监控（9:30–9:35）
- 观察候选股是否站上昨日最高价；
- 若未突破，放弃该股。

第三步：执行买入（9:35–10:00）
- 限价单买入（价格=买一价）；
- 数量为100股整数倍；
- 记录成本价和日期。

第四步：盘中监控（T+1日起）
- 每日收盘前检查：是否盈利≥6%？是否破20日线？是否亏损≥7%？

第五步：尾盘处理（T+3日14:55前）
- 无论盈亏，全部清仓；
- 更新交易日志，准备次日筛选。
