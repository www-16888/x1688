超跌反弹反转策略

1. 选股

选股范围

  • 全市场A股（主板为主）

排除范围

  • ST股及风险警示股票
  • 科创板股票
  • 上市未满250天的次新股
  • 停牌股票
  • 流通市值 < 50亿的股票
  • 新增排除：近一年内出现过重大负面公告（如诉讼、监管处罚等）

核心筛选流程

  • 基本面初筛：ROE > 0，按ROE降序排列取前10
  • 技术面确认：55日均线突破信号验证
  • 超跌特征：长期下跌后的首次反弹
  • 新增要求：剔除资产负债率 > 70%的公司（财务稳健性）

2. 买入条件

技术信号

  • 40日内仅有1-3天突破55日均线
  • 最近3日内有突破55日均线
  • 当前价格站上55日均线
  • 10日内低点为60日内最低点
  • 新增要求：开盘涨幅不超过5%，避免追高

仓位控制

  • 持仓数量：最多5只股票
  • 等权重分配资金
  • 可用现金充足性检查
  • 逐步建仓至满仓

买入方式

  • 尾盘14:45–14:55市价单买入（避开早盘情绪干扰）
  • 按100股整数倍下单

3. 卖出条件

技术卖出

  • 日内高点回撤5%
  • 单日下跌5%
  • 跌破55日均线2%
  • 新增条件：持有满3周仍未上涨，则强制卖出（避免长期滞涨）

止损卖出

  • 总亏损达到5%
  • 任何卖出信号触发立即执行

盘中监控

  • 每日收盘后检查是否跌破20日均线，若连续两日破线，则次日开盘卖出（趋势破坏信号）

4. 仓位管理

持仓数量

  • 最多5只股票（保持分散投资）

资金分配

  • 严格的等权重分配（每只股票约20%仓位）
  • 新增资金管理：未使用资金买入货币ETF（511880），享受T+0流动性与收益

调仓频率

  • 日度信号检查，每周一调仓

仓位控制

  • 接近满仓运作，但保留部分现金应对突发情况

5. 风控管理

质量风控

  • 排除ST、科创板、次新股等风险股票
  • 新增排除：近期有重大负面公告的公司

流动性风控

  • 排除停牌股票
  • 新增要求：日成交额 < 1亿元的股票不参与（确保流动性）

技术风控

  • 多重卖出信号监控
  • 新增要求：每日收盘后检查是否跌破20日均线，若连续两日破线，则次日开盘卖出（趋势破坏信号）

止损风控

  • 5%硬性止损线

分散风控

  • 5只股票分散投资，避免集中持股带来的风险

6. 交易频率

选股计算

  • 开盘前

买入操作

  • 开盘09:30（改为尾盘14:45–14:55）

卖出操作

  • 尾盘14:55

持仓监控

  • 持续跟踪

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每周日晚）

  • 获取全市场股票池
  • 执行基本面和技术面筛选
  • 确定10只候选股票
  • 记录选股结果

第二步：开盘前筛选（周一09:00）

  1. 获取全市场A股，排除风险股票
  2. ST、科创板、次新股、停牌、EPS为负、ROE < 0、资产负债率 > 70%
  3. 选择ROE > 0的股票，按ROE降序排列取前10
  4. 确认55日均线突破信号
  5. 记录最终10只候选股票

第三步：尾盘买入（周一14:45–14:55）

  1. 检查持仓数量是否不足5只
  2. 计算每只股票分配资金
  3. 按等权重执行买入
  4. 确保现金充足性
  5. 若周一开盘涨幅超过5%，则暂缓买入，等待回调后再入场

第四步：盘中监控（每日10:30、13:00、14:00）

  1. 持续跟踪持仓价格
  2. 监控技术指标变化
  3. 准备卖出信号判断
  4. 新增监控：每日收盘后检查是否跌破20日均线，若连续两日破线，则次日开盘卖出

第五步：尾盘卖出（14:55）

  1. 检查所有持仓卖出条件
  2. 执行触发信号的卖出操作
  3. 记录卖出原因和结果

第六步：盘后总结（收盘后）

  1. 分析当日交易记录
  2. 计算账户收益率
  3. 总结策略表现
  4. 优化参数设置

问财筛股步骤

📌 完整版问财选股语句（用于每周一盘前筛选）

非ST，非*ST，非科创板，上市时间大于250天，流通市值大于50亿，近1日成交额大于1亿元，ROE大于0，资产负债率小于70%，股价上穿55日均线，10日内最低价等于60日内最低价，收盘价大于55日均线

💡 说明： - “股价上穿55日均线” ≈ 最近1-3日首次站上均线（问财默认为当日或近期） - “10日内最低价等于60日内最低价” → 精准捕捉“60日新低后反弹”信号 - 问财会自动排除停牌股（因无成交数据）

问财实操建议

1. 使用时机

  • 每周一早盘前（9:00–9:25） 执行问财筛选
  • 输出结果后，人工检查：
  • 是否真为“首次突破”（看K线图确认）
  • 开盘涨幅是否 >5%（若>5%，跳过该股）

2. 排序与选股

  • 问财结果默认按相关性排序，建议手动按 ROE降序 或 跌幅排序（60日跌幅）
  • 选出 10只候选股，再根据流动性、行业分散等人工定5只

3. 无法自动化的部分（需人工）

条件 处理方式
40日内突破次数 ≤3次 查K线图，看55日均线穿越次数
开盘涨幅 ≤5% 盘中观察，若超5%则放弃当日买入
重大负面公告 用“个股F10→重大事项”快速浏览

总结：问财语句清单
