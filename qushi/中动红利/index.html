中小盘动态调仓红利策略

1. 策略分类和核心逻辑

  • 一级分类：基本面量化策略
  • 二级分类：中小盘动态调仓红利策略
  • 核心逻辑：基于中小板指数成分股，通过基本面筛选、红利因子增强、动态仓位调整和严格风控，在特定月份空仓避险，追求稳健的超额收益。

2. 选股条件

  • 选股范围：中小板指数（399101.XSHE）成分股
  • 排除范围：
  • 科创板、创业板、北交所股票
  • ST股、退市标签股
  • 上市未满375天的新股
  • 当日停牌、涨跌停股票
  • 基本面筛选：
  • 盈利能力：归母净利润 > 0，净利润 > 0，ROE > 0，ROA > 0
  • 营收规模：营业收入 > 1亿元
  • 市值范围：10亿 ≤ 总市值 ≤ 10000亿
  • 股价限制：股价 ≤ 50元
  • 红利筛选：
  • 审计过滤：近三年无不合格审计意见
  • 分红要求：排除"不分配不转增"的股票
  • 特殊月份：在设定月份优先选择高分红率股票
  • 核心排序：按总市值从小到大排序

3. 买入条件

  • 调仓频率：每周二调仓一次
  • 买入前提：非空仓月份（1月、4月为空仓月份）
  • 买入规则：
  • 先卖出不再符合条件的持仓
  • 按等分资金买入新入选股票
  • 数量限制：动态调整持仓数量（3-6只）
  • 空仓配置：空仓月份持有货币ETF（511880）

4. 卖出条件

  • 定期调仓：每周评估，不在选股池中即卖出
  • 涨停保护：昨日涨停股，今日涨停打开立即卖出
  • 止盈止损：
  • 个股止盈：收益率 ≥ 100%时卖出
  • 个股止损：跌幅 ≥ 9%时卖出
  • 市场止损：中小板指数跌幅 ≥ 5%时清仓
  • 空仓机制：1月、4月强制清仓股票持仓

5. 仓位管理

  • 持仓数量：动态调整3-6只股票
  • 调整逻辑：基于中小板指数10日乖离率
  • 乖离率 ≥ 5%：持有3只
  • 2% ≤ 乖离率 < 5%：持有3只
  • -2% ≤ 乖离率 < 2%：持有4只
  • -5% ≤ 乖离率 < -2%：持有5只
  • 乖离率 < -5%：持有6只
  • 资金分配：等分可用资金到每只新买入股票
  • 补仓机制：盘中因止损卖出后及时补仓

6. 风控管理

  • 月份风控：1月、4月强制空仓规避风险
  • 个股风控：严格的财务指标和审计意见筛选
  • 价格风控：50元股价上限控制个股风险
  • 交易风控：涨停板跟踪和及时止盈止损
  • 市场风控：基于指数表现动态调整持仓数量

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每个交易日9:05）

  1. 持仓分析：
  2. 检查当前持仓情况
  3. 识别昨日涨停的持仓股

  4. 更新涨停股票监控列表

  5. 月份判断：

  6. 检查当前是否为空仓月份（1月、4月）
  7. 确定当日交易信号

第二步：选股执行（每周二10:00）

  1. 基础筛选：
  2. 获取中小板指数成分股
  3. 排除ST、次新、停牌、涨跌停股票

  4. 筛选市值10-10000亿的股票

  5. 财务筛选：

  6. 筛选净利润、营收、ROE等财务指标达标股票

  7. 排除近三年有不合格审计意见的股票

  8. 红利增强：

  9. 在设定月份优先选择高分红率股票
  10. 平日排除年度不分红的股票

  11. 按市值从小到大排序

  12. 动态调仓：

  13. 计算中小板指数10日乖离率
  14. 根据乖离率确定持仓数量（3-6只）
  15. 选择最终持仓股票

第三步：盘中监控（每个交易日10:00）

  1. 止损执行：
  2. 检查个股是否触及止盈止损线
  3. 检查市场是否触发大跌止损
  4. 执行止损卖出操作

第四步：盘中交易（每个交易日14:00）

  1. 涨停监控：
  2. 监控昨日涨停股是否打开涨停

  3. 涨停打开的股票立即卖出

  4. 补仓操作：

  5. 检查因止损造成的空仓
  6. 及时补足股票持仓至目标数量
  7. 剩余资金买入货币ETF

第五步：尾盘清理（每个交易日14:50）

  1. 空仓执行：
  2. 在空仓月份清仓所有股票持仓
  3. 确保只持有货币ETF
  4. 为下周调仓做准备

第六步：纪律维护

  1. 每周执行：
  2. 严格执行周二调仓纪律
  3. 坚决按月份规则进行空仓

  4. 动态调整持仓数量

  5. 风控坚持：

  6. 不因市场波动改变空仓规则
  7. 严格执行止盈止损纪律
  8. 保持红利因子筛选标准

交易成本考虑： - 买入佣金：0.025% - 卖出佣金：0.025% - 印花税：0.1% - 最低佣金：5元/笔 - 滑点：0.03%

风险提示： 该策略在1月、4月强制空仓，可能错过上涨行情。中小盘股票波动性较大，虽然通过基本面筛选控制质量，但仍需关注市场风格变化。红利因子在特定市场环境下可能失效，需持续监控因子有效性。动态调仓机制基于历史波动率，在极端市场情况下可能放大风险。
