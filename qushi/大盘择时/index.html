大盘择时价值策略（散户实操优化版 · A股合规版）

1.策略的风险等级、分类和策略核心逻辑

  • 风险等级：R3（稳健偏积极型）
  • 策略分类：大盘择时策略 / 价值轮动策略
  • 策略核心逻辑：通过沪深300指数月线MACD判断牛熊市（避免盘中噪音），在牛市月持有高ROE、低PB、大市值的价值股，在熊市月转持货币ETF避险。每月仅调仓1次，所有选股条件均可通过同花顺问财自然语言一键筛选，完全符合A股T+1、交易时段、100股整数倍等规则，适合普通散户每月花10分钟操作。

2.选股条件

  • 选股范围：A股主板（60/00开头）
  • 排除范围：
  • 创业板（300）、科创板（688/689）、北交所（4/8开头）
  • ST、*ST、退市股
  • 停牌股

  • 上市不满300天的新股

  • 价值筛选条件（全部可用问财实现）：

  • 财务质量：ROE > 12%（替代ROA，问财支持更好）
  • 估值安全：市净率 PB < 1.2 且 > 0
  • 市值门槛：总市值 > 500亿元
  • 流动性保障：近20日日均成交额 > 1亿元

✅ 同花顺问财语句（每月1日9:00直接复制使用）：
主板股票，非ST，上市天数大于300，ROE大于12，市净率小于1.2且大于0，总市值大于500亿，近20日日均成交额大于1亿

📌 说明：
- 用 ROE（净资产收益率） 替代 ROA，问财数据更稳定；
- PB<1.2 比 PB<1 更宽松，避免牛市无票可选；
- 日均成交额 替代 Beta 系数（散户无法获取），大市值+高成交天然低波动；
- 结果通常为10–30只，按ROE从高到低取前5只即可。

3.买入条件

  • 买入时机：每月第一个交易日 9:35–10:00（避开集合竞价）
  • 买入前提：
  • 大盘信号：查看沪深300指数月K线，MACD柱状体为红色（DIFF > DEA） → 视为牛市；

  • 若MACD为绿色（熊市）→ 不买股票，全仓持有货币ETF。

  • 买入方式：

  • 等分资金：将可用资金平均分配到5只股票；
  • 买入顺序：按问财结果中ROE从高到低排序，取前5只；
  • 数量要求：每只买入100股整数倍（如200股、500股）；

  • 若结果不足5只：有多少买多少，不强行凑数。

  • 替代投资：

  • 熊市月：全部资金买入 货币ETF，T+0申赎，收益稳定；
  • 不使用沪深300ETF轮动（简化操作）。

4.卖出条件

  • 大盘切换卖出：
  • 若下月首个交易日发现沪深300月线MACD转绿 → 当日10:00前清仓所有股票，转买货币ETF。
  • 月度调仓卖出：
  • 每月1日，卖出上月持仓中未进入新选股前5名的股票；
  • 保留仍在前5名的股票（减少交易频率）。
  • 涨停处理：
  • 若持仓股当日涨停 → 继续持有（不卖）；
  • 不因“昨日涨停”而特殊操作（避免T+1混淆）。
  • 强制清仓：
  • 若某股被ST或停牌 → 次日开盘立即卖出。

✅ 完全符合T+1：所有卖出均为上月或更早买入的股票，可当日卖出。

5.仓位管理

  • 持仓数量：牛市月最多5只，熊市月0只（全仓货币ETF）
  • 仓位比例：
  • 牛市：5只股票等权重，每只≈20%资金；
  • 熊市：100%持有货币ETF；
  • 不留现金，资金全额利用。
  • 调仓频率：每月1次（仅在每月第一个交易日操作）
  • 现金管理：卖出股票资金当日买入货币ETF（若熊市）或新股票（若牛市）

6.风控管理

  • 大盘风控：以月线MACD为唯一择时信号，避免日线噪音；
  • 个股风控：
  • 高ROE（>12%）保障盈利质量；
  • 低PB（<1.2）提供估值安全边际；
  • 大市值（>500亿）+高成交（>1亿）天然低波动；
  • 流动性风控：排除小盘股、ST股、新股；
  • 操作风控：每月仅1次调仓，大幅降低交易错误率；
  • 极端风控：若大盘单月暴跌>10%，下月自动切换至货币ETF。

7.手动实操步骤（纯人工交易版）

第一步：盘前准备（每月第一个交易日 9:00–9:15）
1. 打开同花顺，输入 “399300” 进入沪深300指数K线；
2. 切换为 “月K线”，查看MACD是否为红色（DIFF > DEA）；
3. 若为红色 → 准备选股；若为绿色 → 全仓买入511880，跳过选股。

第二步：基本面筛选（9:15–9:25）
1. 打开同花顺问财，粘贴上述问财语句；
2. 点击“问一下”，等待结果；
3. 按“ROE”列从高到低排序，取前5只（不足5只则全选）；
4. 记录股票代码、名称、ROE值。

第三步：月度调仓（9:35–10:00）
1. 卖出：上月持仓中不在新前5名的股票；
2. 买入：用等分资金买入新选出的5只股票（100股整数倍）；
3. 若为熊市：卖出所有股票，买入货币ETF。

第四步：盘中监控（10:00–14:55）
- 无需盯盘！仅需确认委托成交即可。

第五步：涨停检查（无需操作）
- 持仓股涨停则持有，不因涨停而卖出（避免T+1误解）。

第六步：尾盘整理（15:00）
- 核对持仓是否为最新5只（或全仓货币ETF）；
- 截图保存当月选股结果。

第七步：盘后复盘（可选）
- 每月末回顾：
- 沪深300月线MACD是否准确？
- 价值股是否跑赢指数？
- 是否有更好参数（如PB<1.0）？

关键执行要点：
- 只看月线MACD，不看日线（避免频繁切换）；
- 每月只操作1次，9:35–10:00完成；
- 问财语句一键出结果，无需计算ROA/Beta；
- 熊市全仓货币ETF，零风险避险。
