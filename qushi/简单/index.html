高股息小市值策略（散户简化版）

1. 策略分类和核心逻辑

  • 一级分类：基本面+技术面复合策略
  • 二级分类：高股息低股价策略（简化版）
  • 核心逻辑：
    每月从分红稳定、股价便宜（<9元）、市值适中、非高风险板块的股票中，选出10只“看得懂、买得起、有分红”的股票，等权重持有，每月调仓一次。
    不追求暴利，只求“稳稳的幸福”。

✅ 简化重点：
- 用“股息率 > 3%”代替“全市场前25%”（人工可查阈值）
- 用“市值 < 200亿”代替“按市值排序取最小”（避免极小市值流动性陷阱）
- 聚焦主板（60/00开头），自动排除科创板/北交所

2. 选股条件（人工可执行）

✅ 选股范围：

  • 仅限主板A股：股票代码以 60（沪市）或 00（深市）开头
    → 自动排除科创板（688）、创业板（300）、北交所（8）

✅ 排除范围（一眼可见）：

  • ST / *ST 股票（名称带“ST”）
  • 当日涨停 / 跌停 / 停牌（开盘前行情可见）
  • 股价 ≥ 9元（直接过滤）

✅ 三大筛选标准（全部可在F10查看）：

条件 标准 查看位置
高股息 近12个月股息率 ≥ 3% 同花顺F10 → “分红融资”
小市值 总市值 < 200亿元 F10 → “公司概况” 或 行情界面
低股价 当前股价 < 9元 行情列表直接可见

💡 为什么选3%？
A股中股息率 > 3% 的公司约占前30%，既保证“高分红”，又避免因门槛过高导致样本太少。

3. 买入条件

  • 调仓时间：每月第一个交易日 9:30
  • 持仓数量：固定 10只股票
  • 买入方式：
  • 等权重分配（总资金 ÷ 10）
  • 优先选择行业分散的股票（避免全买银行或全买地产）
  • 若某只股票涨停无法买入，则跳过，从备选中补足10只

4. 卖出条件

  • 常规卖出：调仓日卖出不在新名单中的股票
  • 涨停保护：
  • 若持仓股昨日涨停，今日继续涨停 → 持有
  • 若今日打开涨停 → 14:00前卖出
  • 止损纪律：单只个股亏损达 -15%，立即止损（保护本金）

5. 仓位管理

  • 总仓位：通常满仓（除非大盘极度高估）
  • 单一个股：10%（严格等权）
  • 调仓频率：每月1次（降低摩擦成本）
  • 行业分散：尽量覆盖3个以上行业（如银行、电力、交运、化工、公用事业等）

6. 风控管理

  • 质量风控：只选有连续分红记录的公司（F10查“分红年度”）
  • 流动性风控：市值 > 30亿（避免“仙股”），日成交 > 3000万元
  • 价格风控：股价 < 9元，降低单笔风险
  • 情绪风控：不追热点、不炒题材、不碰“看不懂”的公司

7. 手动实操步骤（纯人工版）

📅 每月第一个交易日操作流程（9:00–9:30）

第一步：用问财快速初筛（9:00–9:10）

在 同花顺i问财（https://www.iwencai.com）输入：

主板股票，非ST，股价<9元，总市值<200亿，股息率>3%，上市满1年，近3年有分红

✅ 结果通常为50~150只股票，导出或截图。

第二步：人工精筛（9:10–9:25）

对问财结果，逐个检查F10： 1. 是否连续分红？（近3年至少分2次）
2. 是否有大额负债或亏损？（看“主要财务指标”）
3. 所属行业是否稳健？（优先选：银行、电力、高速、港口、煤炭、家电等）
4. 当日是否可交易？（非涨跌停、非停牌）

选出10只，按“行业分散 + 股息率高 + 市值适中”排序。

第三步：调仓执行（9:30）

  1. 卖出：不在新名单的持仓（涨停股除外）
  2. 买入：等分资金买入新10只
  3. 记录：在Excel登记股票、成本、行业、股息率

第四步：盘中监控（14:00，仅需1分钟）

  • 检查昨日涨停的持仓是否开板 → 开板即卖

8. 通达信 / 同花顺友好支持

✅ 问财选股语句（直接复制）：

主板，非ST，股价<9元，总市值<200亿，股息率>3%，近3年有分红，上市满1年

9. 风险提示与优化建议

⚠️ 风险：

  • 价值陷阱：高股息但业绩下滑（如某些地产、零售股）→ 需人工排除
  • 小盘流动性差：极端行情下难卖出 → 控制单只仓位≤10%
  • 牛市跑输：小盘价值股在成长牛市中可能滞涨 → 可搭配大盘判断（如上证>3400时减仓）

✅ 优化建议：

  • 加入“ROE > 5%”：在F10中看“净资产收益率”，排除亏损或低效公司
  • 避开“分红后大跌”股：若刚分红完且股价破位，暂缓买入
  • 设置大盘过滤器：上证指数 < 2800 时满仓，> 3400 时仓位 ≤ 60%
