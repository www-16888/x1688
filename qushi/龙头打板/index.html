龙头股打板策略

1. 策略分类和核心逻辑

  • 一级分类：技术面+情绪面策略
  • 二级分类：龙头股打板策略
  • 核心逻辑：基于市场情绪周期理论，通过识别连板龙头股票，在情绪上升周期中参与最强标的，利用龙头股的溢价效应和资金接力特征，追求短期超额收益。

2. 选股条件

  • 选股范围：全市场A股
  • 排除范围：
  • 科创板、北交所股票
  • ST股及其他退市标签股
  • 上市未满50天的次新股
  • 当日停牌股票

核心筛选标准： - 涨停要求：当日涨停股票 - 连板要求：连续涨停天数最多 - 龙头定义：市场中连板数最高的股票 - 数量限制：最多持有10只龙头股

技术面筛选： - 因子筛选：使用5日平均换手率等因子进一步优选 - 排序方式：按因子值排序选择最优标的 - 黑名单机制：排除前日买入失败的股票

3. 买入条件

  • 交易频率：每日调仓
  • 买入规则：
  • 识别当日所有涨停股票
  • 计算每只股票的连板天数
  • 选择连板天数最高的龙头股
  • 用技术因子进一步筛选
  • 等权重分配资金买入
  • 买入方式：
  • 开盘未涨停：市价单立即买入
  • 开盘涨停：限价单排队买入
  • 执行时间：开盘后立即执行

4. 卖出条件

  • 止损卖出：
  • 持仓出现亏损立即止损
  • 避免亏损扩大
  • 条件卖出（14:50后）：
  • 不涨停的持仓股
  • 持有超过2天
  • 或已出现亏损
  • 特殊情况：涨停股继续持有

5. 仓位管理

  • 持仓数量：最多10只股票
  • 资金分配：严格的等权重分配
  • 调仓频率：每日调仓，紧跟市场变化
  • 风险控制：分散投资多只龙头股

6. 风控管理

  • 严格止损：出现亏损立即卖出
  • 时间风控：持有时间不超过2天（特殊情况除外）
  • 流动性风控：排除一字涨停股，确保可买入
  • 黑名单风控：买入失败股票次日排除
  • 分散风控：10只股票分散投资

7. 情绪周期监控

  • 情绪指标：市场最高连板数
  • 周期判断：连板数突破前高为上升周期
  • 情绪记录：持续跟踪市场情绪变化
  • 周期应用：在不同周期采用不同策略

8. 手动实操步骤（纯人工交易版）

第一步：盘前准备（9:01）

  1. 基础筛选：
  2. 获取全市场股票列表
  3. 排除科创、北交、ST、次新、停牌股票

第二步：涨停识别

  1. 涨停股筛选：
  2. 识别前日所有涨停股票
  3. 排除一字涨停股（确保可买入）

第三步：连板计算

  1. 连板分析：
  2. 计算每只涨停股的连续涨停天数
  3. 识别市场中连板数最高的龙头股
  4. 记录市场最高连板数（情绪指标）

第四步：因子筛选

  1. 技术优选：
  2. 使用5日平均换手率等因子
  3. 对龙头股进行排序筛选
  4. 排除前日买入失败股票

第五步：开盘买入（9:30）

  1. 买入执行：
  2. 对未涨停股票使用市价单
  3. 对涨停股票使用限价单排队
  4. 等权重分配资金到目标股

第六步：盘中监控（10:30、13:00、14:00）

  1. 止损检查：
  2. 监控持仓股是否出现亏损
  3. 执行严格止损纪律

第七步：尾盘卖出（14:50后）

  1. 条件卖出：
  2. 卖出不涨停的持仓股
  3. 卖出持有超过2天的股票
  4. 卖出已出现亏损的股票

第八步：收盘总结（15:02）

  1. 持仓分析：
  2. 记录当日买入情况
  3. 更新买入失败黑名单
  4. 分析策略执行效果

策略理论基础： - 龙头效应：市场中最强股票具有持续溢价 - 情绪周期：市场情绪存在明显的周期特征 - 资金接力：龙头股具有资金接力的特征 - 强者恒强：趋势延续性在龙头股中表现明显

交易执行要点： - 快速反应：开盘立即执行，抢占先机 - 灵活下单：根据涨停情况选择下单方式 - 严格纪律：止损和卖出条件坚决执行 - 持续优化：根据市场变化调整参数

风险提示： 该策略为高风险高收益策略，在情绪退潮期可能面临较大回撤。打板策略对执行速度和交易纪律要求极高，不适合普通投资者。龙头股波动性极大，单日可能出现大幅震荡。策略依赖市场情绪的持续性，在监管政策变化或市场风格切换时可能失效。需要密切监控市场情绪变化，及时调整策略。
