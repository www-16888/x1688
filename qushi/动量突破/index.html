动量突破趋势策略

1. 选股

✅ 选股范围

  • 全市场A股（主板为主）

❌ 排除股票（同花顺“条件选股”中勾选）

  • ST/*ST/退市股
  • 科创板（688开头）、创业板（300开头）
  • 流通市值 < 50亿元（F10可查）
  • 当日涨停或跌停股（盘后自动过滤）

🔍 技术信号（肉眼观察即可）

信号 判断方式（同花顺K线图）
1. 突破前高 股价今日收盘价 > 近20日最高价（看K线前20根，找最高点）
2. 站稳20日均线 MA20 向上，且今日K线收在MA20上方（调出MA指标）
3. 放量 今日成交量柱 明显高于前5日平均（肉眼判断，或看“量比 > 1.4”）
4. 动量初显 过去5日涨幅 3%~10%（避免已大涨股）
→ 同花顺输入“5日涨幅”排序即可
5. MACD辅助 MACD在0轴附近或上方，红柱开始放大（非必须，加分项）

✅ 无需计算ROC或Momentum！用“5日涨幅+放量+突破”替代动量指标。

2. 买入条件

  • 买入时机：T+1日尾盘 14:45–14:55（非早盘！）
  • 买入前提：
  • T日满足上述5项信号
  • T+1日未高开 > 5%（避免追高）
  • T+1日股价在MA10上方震荡或小幅回调
  • 仓位控制：单只 ≤ 20% 总资金
  • 最大持仓：同时持有 ≤ 3只
  • 买入方式：市价单（尾盘流动性好）

⚠️ 关键改进：放弃“9:35追突破”，改为次日尾盘低吸，大幅提升胜率。

3. 卖出条件

情况 操作
止盈 持有期间累计涨幅 ≥ 12%，次日冲高卖出
止损 收盘价跌破MA10，次日开盘卖出
时间止损 持有满5个交易日仍未达12%，第6日开盘清仓
大盘破位 上证指数收盘跌破20日线，所有持仓次日清仓

✅ 不再依赖Momentum转负（散户难判断），改用MA10破位这一清晰信号。

4. 仓位与风控（

项目 规则
单票仓位 ≤ 20%
最大持仓 3只
空仓条件 上证指数 < MA20 或 市场连板高度 ≤ 2板
流动性要求 日成交额 ≥ 3亿元（F10可见）
组合风控 账户总回撤 ≥ 5%，暂停开仓1周

5. 手动实操步骤

🕒 第一步：盘后选股（15:30–16:00）

非ST，非科创板，非创业板，主板，流通市值大于50亿，收盘价创20日新高，5日涨幅介于3%到10%，量比大于1.4，股价站上20日均线，20日均线向上，所属行业近3日有2只以上涨停股

🕒 第二步：次日尾盘决策（14:45–14:55）

  1. 查看自选股：
  2. 是否未高开（<5%）？
  3. 是否在MA10上方？
  4. 所属板块是否仍有热度？
  5. 选1–2只最强标的，市价买入 ≤20%仓位

🕒 第三步：持有与卖出（T+2日起）

  • 每日收盘检查：是否破MA10？是否达12%？是否满5天？
  • 按第4节规则执行

🕒 第四步：每周复盘（周日晚）

  • 统计：胜率、平均持有天数、最大回撤
  • 若胜率 < 40%，暂停并反思
