首板+二板龙虎榜挖掘策略

1. 策略分类和核心逻辑

  • 一级分类：事件驱动策略
  • 二级分类：龙虎榜+涨停板策略
  • 核心逻辑：基于龙虎榜机构买入信号，结合首板或二板涨停特征，挖掘有机构资金介入的强势股，通过高频交易捕捉短线机会。

2. 选股条件

  • 选股范围：全市场A股
  • 排除范围：
  • 科创板、北交所、创业板、B股
  • ST股及其他退市标签股
  • 当日停牌、涨跌停股票

核心筛选标准：

  • 龙虎榜要求：
  • 前日登上龙虎榜
  • 净买入额 > 0
  • 买入占比 > 4%
  • 涨停要求：
  • 首板或二板涨停
  • 昨日收盘价等于涨停价
  • 成交量要求：
  • 最近3日平均成交量是前17日的4-6倍
  • 成交量显著放大但不过度

附加筛选条件：

  • 概念热度：所属概念板块至少有2只股票
  • 换手率：换手率 < 50%
  • 流通市值：按流通市值从大到小排序

3. 买入条件

  • 交易频率：每日交易
  • 买入规则：
  • 开盘价 > 昨日涨停价4%以上
  • 开盘未涨停
  • 最多持有1只股票
  • 全仓买入
  • 买入方式：
  • 使用限价单买入
  • 按100股整数倍下单
  • 执行时间：9:26（开盘前）

4. 卖出条件

  • 止损卖出：
  • 未涨停即卖出
  • 严格执行成本价止损
  • 卖出时机：全天多个时间点监控
  • 11:00, 11:29, 13:00, 13:30
  • 14:00, 14:30, 14:45, 14:55
  • 卖出逻辑：不涨停就卖出，锁定收益

5. 仓位管理

  • 持仓数量：最多1只股票
  • 资金分配：全仓单只股票
  • 调仓频率：每日可能调仓
  • 风险控制：严格的止损纪律

6. 风控管理

  • 流动性风控：排除涨跌停股票
  • 质量风控：排除ST、次新等问题股票
  • 集中度风控：单只股票全仓，高风险高收益
  • 止损风控：不涨停立即卖出，控制亏损
  • 概念风控：要求有板块效应，避免孤军奋战

7. 龙虎榜逻辑

  • 机构认可：龙虎榜代表大资金关注
  • 买入力度：买入占比体现资金决心
  • 净流入：净买入额显示资金方向
  • 持续性：首板或二板体现启动特征

8. 手动实操步骤（纯人工交易版）

第一步：盘前准备（9:15）

  1. 龙虎榜筛选：
  2. 获取前日龙虎榜数据
  3. 筛选净买入额 > 0的股票
  4. 筛选买入占比 > 4%的股票
  5. 按买入占比从高到低排序

第二步：条件过滤

  1. 基础过滤：
  2. 排除科创、北交、创业、B股
  3. 排除ST、退市标签股

  4. 排除停牌、涨跌停股票

  5. 概念筛选：

  6. 分析每只股票所属概念
  7. 筛选所属概念至少有2只股票的
  8. 确保有板块效应

第三步：首板/二板识别

  1. 涨停分析：
  2. 检查昨日是否涨停
  3. 统计近18日涨停次数

  4. 筛选首板或二板股票

  5. 成交量验证：

  6. 计算最近3日平均成交量
  7. 计算前17日平均成交量
  8. 验证3日量是17日量的4-6倍

第四步：最终排序

  1. 换手率筛选：
  2. 排除换手率 > 50%的股票
  3. 按流通市值从大到小排序
  4. 确定最终备选名单

第五步：开盘确认（9:26）

  1. 买入条件确认：
  2. 检查开盘价 > 昨日涨停价4%
  3. 确认开盘未涨停
  4. 执行全仓买入

第六步：盘中监控（全天多个时点）

  1. 持仓监控：
  2. 检查持仓股是否仍涨停
  3. 如未涨停立即卖出
  4. 严格执行止损纪律

策略逻辑解析： - 龙虎榜信号：机构资金介入的明确信号 - 首板启动：行情启动的早期识别 - 成交量配合：资金进场的客观证据 - 板块效应：热点的持续性和扩散性 - 严格止损：控制风险的纪律性

风险收益特征： - 高风险：全仓单只股票，波动性极大 - 高收益：捕捉强势股连续涨停机会 - 高频交易：每日可能调仓，交易频繁 - 严格纪律：依赖极强的执行纪律

交易成本考虑： - 买入佣金：未明确设置（使用默认） - 卖出佣金：未明确设置（使用默认） - 印花税：未明确设置（使用默认） - 滑点：0%

风险提示： 该策略为极高风险策略，全仓单只股票可能导致巨大亏损。龙虎榜数据可能存在滞后性，大资金可能已经出货。首板策略在震荡市中成功率较低。策略对执行时机要求极高，稍有延迟可能导致巨大差异。需要投资者有极强的心理承受能力和严格的止损纪律。不建议普通投资者尝试。
