首板低开策略

1. 策略分类和核心逻辑

  • 一级分类：方向性策略
  • 二级分类：事件驱动策略（核心博弈“首板涨停”后“低开”这一特定事件模式）
  • 核心逻辑：捕捉强势股（昨日涨停）在次日因市场情绪或主力洗盘而低开时产生的“预期差”机会。策略假设首板后的低开是短暂的，股价会延续前日的强势而回升，从而在低开时买入，并在当日冲高时卖出，获取短线收益。

2. 选股条件

  • 选股范围：全市场A股（排除科创板、北交所、ST股、停牌股、上市未满250天的新股）。
  • 筛选方法：
  • 基础过滤：过滤掉不符合条件的股票（见选股范围）。
  • 核心事件：筛选出昨日涨停的股票。
  • 排除连板：排除连续涨停（2板及以上）的股票，专注于“首板”。
  • 相对位置：股价处于过去60日内的相对低位（相对位置 rp ≤ 0.5）。
  • 关键信号：今日开盘价相对于昨日收盘价低开3%-4%（开盘涨幅在96%至97%之间）。
  • 更新频率：每日开盘前。

3. 买入条件

  • 触发信号：同时满足以上所有选股条件。
  • 确认方式：在早上9:30，获取到所有股票的真实开盘价后，程序自动计算并确认。
  • 执行时点：上午9:30开盘后立即执行。

4. 卖出条件

  • 止盈条件（上午）：在11:28，如果持仓股的当前价 < 当日涨停价（即未封板）且当前价 > 持仓成本，则全部卖出，锁定利润。
  • 止损/清仓条件（下午）：在14:50，无论盈亏，只要持仓股未封死涨停，则全部卖出，避免隔夜风险。

5. 仓位管理

  • 资金分配：如果筛选出多只符合条件的股票，则等分总资金买入。例如，有3只股票，则每只股票投入总资金的1/3。
  • 加减仓规则：不加仓。每次交易都是全新的组合，前一日的持仓会在卖出条件触发时全部清空。
  • 总仓位控制：策略本身不设总仓位上限，但通过“空仓时才买入”的规则，保证了同一时期只会运行一个投资组合。

6. 风控管理

  • 个股风控：
  • 通过“低开3%-4%”的限制，控制了单笔交易的初始风险。
  • 通过“相对低位”选股，避免追高历史高位的涨停股。
  • 强制在收盘前清仓，规避隔夜不确定性和系统性风险。
  • 组合风控：通过等分资金自动实现分散投资，避免单一个股的黑天鹅风险。
  • 系统风控：排除了ST、停牌、新股等高风险或无法交易的标的。

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每个交易日上午9:15-9:25）

  1. 准备工具：打开你的股票交易软件和记事本。
  2. 生成昨日涨停股列表：
  3. 在软件中查找“昨日涨停”的板块或功能，将列表导出或记录下来。
  4. 手动过滤：从这个列表中，人工剔除：
    • 股票代码以 4， 8 开头，或以 68 开头的（科创板/北交所）。
    • 名称前带有*ST或ST的股票。
    • 上市时间较短（例如少于1年）的新股。
    • 昨日是连续第2个或更多涨停板的股票（需要手动查看K线确认）。

第二步：开盘决策（上午9:25-9:30）

  1. 确认集合竞价结果：集合竞价结束后（9:25），查看经过第一步过滤后的股票池。
  2. 筛选低开股：逐一计算这些股票的开盘涨跌幅：(今日开盘价 / 昨日收盘价) - 1。
  3. 最终买入名单：只保留那些低开3%到4%（即开盘涨跌幅为 -3% 到 -4%）的股票。
  4. 计算买入金额：如果你的总资金是10万元，最终名单有2只股票，则每只股票计划投入5万元。

第三步：盘中执行

  1. 买入操作（9:30）：
  2. 开盘后，立即对最终买入名单里的股票进行买入。
  3. 委托方式：建议使用“限价委托”，价格可以设为买一价或当前价，确保快速成交。

  4. 买入数量：用计划投入金额 / 当前股价，取整后下单。

  5. 卖出操作（严格按时间表执行）：

  6. 上午11:28：检查所有持仓。
    • 如果某只股票当前股价高于你的成本价，并且没有封死涨停，就全部卖出。
  7. 下午14:50：检查所有剩余的持仓。
    • 无论盈亏，只要没有封死涨停，就全部卖出。

第四步：纪律与记录

  1. 严格守时：必须在规定的时间点（9:25, 9:30, 11:28, 14:50）执行相应操作，克服犹豫和贪婪。
  2. 保持空仓：收盘后，你应该处于空仓状态。如果没有股票满足买入条件，则当天不操作。
  3. 做好记录：记录每笔交易的股票、买卖时间、价格、盈亏原因，用于周末复盘。

风险提示：该策略是短线交易，成功率并非100%，可能遇到“低开低走”的情况。手动执行存在延迟和情绪干扰，需严格遵守纪律。
