错杀反弹策略（散户实操优化版）

1. 策略的风险等级、分类和策略核心逻辑

  • 风险等级：R4（积极型）——简化信号、缩短持仓、强化止盈
  • 策略分类：逆向投资策略 / 小盘错杀反弹策略（简化版）
  • 策略核心逻辑：在市场出现明显恐慌情绪（如中证1000指数大跌、跌停股激增）的交易日，筛选超跌、小市值、非ST、有流动性的股票，于盘中低点介入，博弈次日或短期技术性反弹。核心是“看得见的恐慌 + 可买的错杀股 + 快进快出”，避免主观抄底，只做情绪冰点后的反弹。

2. 选股条件（适配普通软件）

  • 选股范围：A股主板 + 创业板（排除科创板688/689、北交所4/8开头、ST/*ST、停牌股、上市<100天新股）
  • 基础池：全市场符合条件的小盘股

简化筛选条件（全部可通过同花顺/通达信实现）：

  1. 市值要求：流通市值 < 30亿元（F10可查，确保小盘弹性）
  2. 价格要求：股价 < 15元（降低单股投入，提高可买性）
  3. 状态排除：非ST、非停牌、今日未跌停开盘
  4. 流动性保障：昨日成交额 ≥ 3000万元（避免无量阴跌股）
  5. 超跌信号：当日跌幅 ≥ 5%（盘中实时筛选，代表被错杀）

✅ 同花顺问财语句（盘中9:45后使用）：
流通市值<30亿，股价<15元，非ST，主板或创业板，上市天数>100，昨日成交额>30000000，今日跌幅>5%，今日未跌停
（注：问财支持“今日跌幅”实时筛选，但需在9:45后使用，避免集合竞价干扰）

✅ 通达信盘中选股公式（9:45后运行）：
```pascal { 错杀反弹 - 盘中筛选 } SMALL_MCAP := FINANCE(40) < 3000000000; { 流通市值<30亿 } PRICE_LT_15 := C < 15; NOT_ST := NAMELIKE('ST')=0 AND NAMELIKE('*ST')=0; MAIN_OR_GEM := CODELIKE('60') OR CODELIKE('00') OR CODELIKE('30'); NOT_NEW := FINANCE(42) > 100; LIQ_YESTER := REF(AMO,1) > 30000000; { 昨日成交额>3000万 } FALL_TODAY := (REF(C,1) - C) / REF(C,1) >= 0.05; { 今日跌幅≥5% } NOT_LIMIT_DOWN := L > REF(C,1) * 0.9; { 未跌停（留余地） }

SMALL_MCAP AND PRICE_LT_15 AND NOT_ST AND MAIN_OR_GEM AND NOT_NEW AND LIQ_YESTER AND FALL_TODAY AND NOT_LIMIT_DOWN; ```

3. 买入条件

  • 触发前提（恐慌日确认）：满足以下任一即视为“恐慌日”：
  • 中证1000指数（000852）当日跌幅 ≥ 3%；或
  • 同花顺问财查“今日跌停家数” > 40家
  • 买入时机：恐慌日当天 9:45–10:30（避开开盘恐慌杀跌，捕捉初步企稳）
  • 买入方式：
  • 从问财/通达信筛选结果中，选择跌幅最大、量能放大（今日成交量 > 昨日50%）的3–5只；
  • 等分可用资金买入（如总资金6万，买3只则各2万）；
  • 限价单买入：价格设为当前买一价或略高0.01元，确保成交；
  • 单笔≥100股，符合交易规则。

4. 卖出条件（明确+可执行）

  • 止盈卖出：
  • 次日冲高卖出：若次日高开或盘中上涨 ≥ 3%，立即卖出；

  • 当日反弹卖出：若买入当日跌幅收窄至 ≤ -2% 或翻红，可部分止盈。

  • 止损卖出：

  • 买入次日继续大跌：若次日收盘价 < 买入价 × 0.96（-4%），收盘前止损；

  • 盘中破位：若买入后继续下跌且无反弹迹象（如10:30仍跌>7%），当日止损。

  • 强制清仓：

  • 最长持有2个交易日（T日买入 → T+2日14:50强制卖出，无论盈亏）

📌 操作提示：
- 设置手机闹钟：T+1日9:30（观察高开）、T+2日14:50（清仓）；
- 不幻想“V型反转”，反弹即卖。

5. 仓位管理

  • 最大持仓数：3只（避免过度分散，聚焦最强错杀标的）
  • 单股仓位：≤ 总资金的35%
  • 总仓位：仅在确认“恐慌日”时操作，非恐慌日空仓
  • 换仓频率：每周最多操作1–2次，避免频繁交易

6. 风控管理

  • 情绪风控：仅在“中证1000跌≥3%”或“跌停>40家”时启动策略
  • 个股风控：通过流通市值<30亿+成交额>3000万+非ST，规避流动性陷阱
  • 时间风控：2日持有上限，防止陷入阴跌
  • 亏损风控：单笔最大亏损控制在4%以内
  • 极端风控：若买入后市场继续崩盘（如中证1000再跌2%），次日开盘全割

7. 手动实操步骤（纯人工+普通软件版）

第一步：恐慌日识别（9:30–9:40）

  1. 打开同花顺，输入 “000852” 查看中证1000指数跌幅；
  2. 或打开问财，输入 “今日跌停家数”，看是否 > 40；
  3. 若满足任一条件，标记为“今日可操作”。

第二步：盘中选股（9:45–10:00）

  1. 在同花顺问财输入筛选语句（见第2节）；
  2. 导出结果（通常10–30只）；
  3. 人工排序：按今日跌幅从大到小，剔除无量阴跌股（今日量 < 昨日50%）；
  4. 选出3只跌幅大、量能放大、K线有下影线的股票。

第三步：买入执行（10:00–10:30）

  1. 计算每只买入金额（总资金 ÷ 3）；
  2. 限价单买入（价格=当前买一）；
  3. 记录：股票、买入价、买入时间。

第四步：持仓监控

  • T+1日9:30：若高开≥1%或冲高≥3%，卖出；
  • T+1日收盘：若收盘价 < 买入价×0.96，准备止损；
  • T+2日14:50：无论盈亏，全部清仓。

第五步：盘后复盘

  • 记录交易日志：
    | 日期 | 是否恐慌日 | 股票 | 买入价 | 卖出价 | 持有天数 | 盈亏% |
  • 每月统计：胜率、平均收益、最大回撤；
  • 连续2次亏损，暂停策略1周。

⚠️ 风险提示（散户特别注意）

  • 该策略仍属高波动短线交易，单笔亏损可达4%–6%；
  • “恐慌日”并非必反弹，若遇系统性风险（如黑天鹅），可能连续下跌；
  • 严禁在非恐慌日强行操作，避免“接飞刀”；
  • 强烈建议先用模拟盘或≤10%资金试运行1个月，熟悉流程后再加大投入。

✅ 一句话总结：
“中证1000跌≥3% + 小市值+跌超5%”是选股核心，“次日冲高即卖+2日清仓”是保命纪律。借助问财实时筛选+人工排序，普通散户可有限执行，但必须严守恐慌信号与止损规则。
