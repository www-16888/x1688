错杀反弹策略

1.策略的风险等级、分类和策略核心逻辑

  • 风险等级：R5（激进型）
  • 策略分类：逆向投资策略 / 小盘股轮动策略
  • 策略核心逻辑：基于市场恐慌情绪识别被错杀的小盘股，通过构建小盘股指数监控市场情绪，在恐慌性下跌后捕捉反弹机会。核心在于"恐慌识别+小盘轮动+及时止盈"，利用市场非理性行为获取超额收益。

2.选股条件

  • 选股范围：中小板指数成分股（399101.XSHE）
  • 基础池：中小板全部股票
  • 筛选条件：
  • 市值筛选：流通市值最小的15只股票（5×3）
  • 状态排除：排除停牌、ST、*ST、退市股票
  • 涨跌停排除：排除涨停开盘和跌停开盘股票
  • 特殊保留：昨日涨停股票优先保留
  • 最终数量：精选5只股票构建组合

3.买入条件

  • 买入时机：周一9:16选股，10:40执行买入
  • 买入条件：
  • 恐慌信号：小盘股指数较近期高点下跌超过22%
  • 市场状态：当前处于恐慌状态（g.panic_today_signal=True）
  • 持仓要求：持仓数量未达到目标数量
  • 买入方式：
  • 等分资金：可用现金平均分配到目标股票
  • 数量限制：严格控制在5只股票以内
  • 顺序买入：按筛选顺序依次买入

4.卖出条件

  • 恐慌止损：小盘股指数较前一日下跌2%立即清仓
  • 涨停管理：
  • 昨日涨停股票：今日涨停打开立即卖出
  • 持续涨停：继续持有观察
  • 时间止盈：持仓超过10天且指数下跌时减仓
  • 风险止盈：小盘股指数较近期高点下跌22%时清仓

5.仓位管理

  • 持仓数量：固定持有5只小盘股
  • 仓位比例：
  • 单一个股：等分资金，集中投资
  • 总仓位：通常接近满仓运作
  • 现金管理：非恐慌期保持现金
  • 动态调整：每周一更新股票池，根据恐慌信号动态调整

6.风控管理

  • 情绪风控：通过小盘股指数实时监控市场情绪
  • 波动风控：小盘股特有的高波动性管理
  • 流动性风控：中小板股票确保基本流动性
  • 技术风控：严格的止损止盈机制
  • 极端风控：恐慌信号的快速响应和仓位控制

7.手动实操步骤（纯人工交易版）

第一步：盘前准备（9:00-9:15） 1. 更新持仓记录，识别昨日涨停股票 2. 计算持有天数，为时间止盈做准备 3. 检查小盘股指数，初步判断市场情绪

第二步：恐慌状态检查（9:15） 1. 计算小盘股指数相对高点跌幅 2. 判断是否处于恐慌状态（跌幅超过22%） 3. 记录当日交易信号（恐慌/正常）

第三步：周度选股（周一9:16） 1. 获取中小板指数成分股 2. 按流通市值从小到大排序，取前15名 3. 排除停牌、ST、涨跌停股票 4. 加入昨日涨停股票并优先排序 5. 最终确定5只目标股票

第四步：日内买入（10:40） 1. 确认恐慌信号仍然有效 2. 检查持仓数量，计算需要买入的股票 3. 等分资金买入目标股票 4. 重置持有天数计数器

第五步：涨停检查（14:00） 1. 检查昨日涨停股票今日表现 2. 涨停打开：立即卖出 3. 继续涨停：保持持有 4. 记录每笔卖出操作

第六步：时间止盈（14:55） 1. 检查持仓天数是否超过10天 2. 检查小盘股指数是否下跌 3. 满足条件：卖出未涨停的持仓股票 4. 不满足条件：继续持有

第七步：恐慌止损（14:55） 1. 计算小盘股指数实时涨跌幅 2. 较前一日下跌2%：立即清仓所有持仓 3. 发送恐慌卖出信号 4. 记录清仓原因和时间

第八步：盘后分析（15:00后） 1. 更新小盘股指数历史数据 2. 分析恐慌信号的准确性 3. 评估选股组合的表现 4. 优化恐慌阈值和止损参数

关键执行要点 - 严格遵循恐慌信号，不主观判断市场底部 - 专注小盘股，利用其高弹性的特点 - 快速响应市场变化，及时止盈止损 - 控制持仓数量，避免过度分散 - 完善指数监控体系，提高信号准确性

风险提示 该策略依赖恐慌情绪的准确识别，在温和下跌市场中可能失效。小盘股的高波动性可能带来较大回撤。恐慌阈值的设定需要根据市场环境动态调整，过度敏感可能导致频繁交易。
