涨停板回调策略（散户实操优化版 · A股合规版）

1. 策略分类和核心逻辑

  • 一级分类：技术面短线策略
  • 二级分类：涨停板回调 + 低吸战法
  • 核心逻辑：在符合A股T+1规则的前提下，筛选近5日内曾涨停、当前回调至支撑位、未破位的小市值主板股，于次日早盘低吸，持有1–3个交易日博弈反弹。核心是“有涨停基因 + 回调不过度 + 位置相对安全”，避免追高，专注可买、可卖、可止损的标的。

2. 选股条件

  • 选股范围：A股主板（60/00开头）
  • 排除范围：
  • 创业板（300开头）、科创板（688/689开头）
  • ST、*ST、退市风险警示股票
  • 停牌股票
  • 流通市值 ≥ 200亿元（排除大盘股）
  • 上市不满375天的次新股

3. 核心筛选流程

第一步：周度基础池准备（每周日晚或周一早）
- 使用问财筛选全市场符合长期条件的股票（非次新、小市值、非ST、主板）

第二步：每日盘前精细筛选（交易日9:00–9:20）
1. 涨停记录：近5个交易日内至少有1次涨停
2. 价格回调：当前股价较涨停日最高价回调10%–25%（不过度杀跌）
3. 未破位：股价仍在20日均线之上（趋势未坏）
4. 量能稳定：昨日成交额 > 3000万元（保障流动性）
5. 开盘温和：集合竞价涨幅在 -2% 至 +3% 之间（无极端情绪）

✅ 同花顺问财语句（直接复制使用）：
主板股票，非ST，上市天数大于375，流通市值小于200亿，近5日有涨停，当前价比近5日最高价低10%到25%，股价高于20日均线，昨日成交额大于30000000

📌 说明：
- 问财支持“近5日有涨停”“流通市值”“20日均线”等自然语言；
- “当前价比近5日最高价低10%到25%”可有效捕捉合理回调；
- 结果通常为5–15只，人工再按集合竞价涨幅排序剔除高开/低开极端股。

4. 买入条件

  • 技术信号：
  • 满足问财筛选结果
  • 集合竞价涨幅 ∈ [-2%, +3%]
  • 未跌停、未一字涨停
  • 买入时机：9:35–10:00（连续竞价开始后，避开开盘波动）
  • 仓位控制：
  • 固定持仓3只（最多）
  • 等权重分配（如总资金6万，每只2万）
  • 单股数量为100股整数倍（符合A股交易单位）
  • 建仓方式：一次性买入，不补仓

5. 卖出条件

  • 主动止盈：
  • 持仓第2日或第3日冲高涨幅 ≥ 6% → 卖出
  • 涨停封板 → 可持有至开板日卖出
  • 强制止损：
  • 买入后任一交易日收盘价 < 买入价 × 0.93（-7%硬止损）
  • 跌破20日均线 → 次日开盘卖出
  • 调仓卖出：
  • 股票不再出现在新一日问财筛选结果中 → 次日卖出
  • 时间清仓：
  • 最长持有3个交易日（T日买入 → T+3日14:55前清仓）

✅ 完全符合T+1：T日买入 → T+1日起可卖出。

6. 仓位管理

  • 固定持仓数量：3只（新手建议1–2只）
  • 等权重分配：每只约30%–35%资金
  • 调仓频率：每日盘前更新候选池，仅替换不合格个股（非全换）
  • 现金管理：无符合标的时空仓，不强行交易

7. 风控管理

  • 质量风控：排除ST、次新、创业板、科创板
  • 流动性风控：成交额 > 3000万 + 主板交易，确保可成交
  • 技术风控：20日均线 + 回调幅度双重过滤
  • 止损风控：-7%硬止损 + 跌破均线离场
  • 分散风控：3只行业尽量不集中（如不全买新能源）

8. 交易频率

  • 周度准备：每周初更新基础池（可选）
  • 每日盘前：9:00用问财筛选当日候选股
  • 开盘交易：9:35–10:00买入，14:55前可卖出（T+1日后）
  • 盘中监控：每日收盘前检查是否触发止损/止盈
  • 盘后总结：记录交易日志，每周复盘

9. 手动实操步骤

第一步：盘前选股（9:00–9:20）
1. 打开同花顺问财，粘贴上述语句；
2. 导出结果，剔除集合竞价涨幅 < -2% 或 > +3% 的股票；
3. 保留3只“回调合理+均线支撑+量能稳定”的标的。

第二步：开盘买入（9:35–10:00）
1. 限价单买入（价格=当前买一价）；
2. 数量为100股整数倍；
3. 记录：股票、买入价、日期。

第三步：持仓监控（T+1日起）
- 每日检查：是否盈利≥6%？是否破20日线？是否亏损≥7%？
- 若在新问财池中消失 → 次日卖出。

第四步：卖出执行（T+1至T+3日 9:30–14:55）
- 按规则止盈/止损/调仓；
- T+3日14:55前强制清仓。

第五步：盘后复盘（15:00后）
- 更新交易记录；
- 评估策略有效性；
- 优化候选股选择标准。

策略优势：
- 选股条件全部可通过问财自然语言实现；
- 买入卖出时点明确，符合A股T+1、100股整数倍、交易时段等规则；
- 止损止盈量化清晰，避免情绪干扰；
- 适合兼职散户每日10分钟操作。

风险提示：
- 小市值股波动大，单笔亏损可能达7%；
- 熊市中回调股可能持续阴跌，需严格止损；
- 严禁在大盘暴跌日（如上证跌>2%）操作；
- 建议先用模拟盘或≤10%资金试运行1个月。
