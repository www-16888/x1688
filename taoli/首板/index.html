首板混合策略（散户实操优化版）

1. 策略的风险等级、分类和策略核心逻辑

  • 风险等级：R4（积极型）——较原版略降，因简化条件、减少持仓、强化止损
  • 策略分类：事件驱动策略 / 首板混合策略（简化版）
  • 策略核心逻辑：聚焦昨日首次涨停的股票，在次日开盘后根据低开、高开、弱转强三种典型模式中的可识别信号，选择1–2只最符合条件的标的介入。利用涨停惯性博弈当日冲高收益，通过尾盘强制清仓+跌破均线止损控制风险。核心是“看得见、算得清、买得进、卖得出”。

2. 选股条件（适配普通软件）

  • 选股范围：A股主板 + 创业板（排除科创板688/689、北交所4/8开头、ST/*ST、停牌股、上市<50天新股）
  • 基础池：昨日首次涨停（昨日涨停，前日未涨停）

三种模式（简化可执行版）：

A. 首板低开模式（推荐优先）

  1. 价格位置：当前价 ≤ 60日最高价 × 0.5（即处于60日相对低位）
  2. 成交额：昨日成交额 ≥ 1亿元
  3. 开盘幅度：低开3%–4.5%（开盘价 ∈ [昨日收盘价×0.955, 昨日收盘价×0.97]）

B. 首板高开模式（谨慎参与）

  1. 成交额：昨日成交额 5亿–20亿元
  2. 市值范围：总市值 70亿–520亿元（F10可查）
  3. 开盘幅度：高开0%–6%
  4. 量能确认：集合竞价成交量 > 昨日同期（9:15–9:25）的3倍（可用“量比 > 3”近似判断）

C. 弱转强模式（需K线辅助）

  1. 前日K线：昨日涨停，前日收阴线（收盘价 < 开盘价）
  2. 短期涨幅：近3日累计涨幅 ≤ 28%（大致判断：未连续大涨）
  3. 成交额：昨日成交额 3亿–19亿元
  4. 开盘表现：高开 -2% 至 +6%，且量比 > 3

✅ 同花顺问财语句（盘后筛选“昨日首板池”）：
昨日涨停，前日未涨停，非ST，主板或创业板，上市天数>50，代码不以688、689、4、8开头
（注：三种模式的具体条件需盘中人工核对，无法全自动）

✅ 通达信首板识别公式（保存为选股）：
pascal ZT_YESTER := C/REF(C,1) >= 1.095 AND C = H; { 昨日涨停 } NOT_ZT_BEFORE := REF(C,2)/REF(C,1) < 1.095; { 前日未涨停 } NOT_ST := NAMELIKE('ST')=0 AND NAMELIKE('*ST')=0; MAIN_OR_GEM := CODELIKE('60') OR CODELIKE('00') OR CODELIKE('30'); NOT_NEW := FINANCE(42) > 50; ZT_YESTER AND NOT_ZT_BEFORE AND NOT_ST AND MAIN_OR_GEM AND NOT_NEW;

3. 买入条件

  • 买入时机：9:26–9:35之间（集合竞价结束后，避开瞬时波动）
  • 买入方式：
  • 最多选2只股票（大幅降低操作难度）
  • 等分可用资金（如总资金6万，买2只则各3万）
  • 限价单买入：价格设为开盘价或当前买一价（避免市价单滑点）
  • 单笔≥100股（符合交易规则）
  • 买入顺序：优先低开模式 → 弱转强 → 高开模式（按风险从低到高）

4. 卖出条件（明确+可执行）

  • 止盈卖出：
  • 当日涨停且封单稳定 → 可持有至尾盘，但14:50仍未封死则卖出

  • 盘中盈利 ≥ 3% 且出现冲高回落（5分钟内跌1.5%） → 卖出

  • 止损卖出：

  • 盘中跌破5日均线（看分时图叠加5日线）→ 立即止损

  • 亏损 ≥ 4% → 当日或次日开盘坚决止损

  • 强制清仓：

  • 14:50：所有未涨停的持仓全部卖出（无论盈亏）

📌 工具支持：
- 同花顺分时图可叠加“5日均线”；
- 设置手机闹钟：14:50 提醒清仓。

5. 仓位管理

  • 最大持仓数：2只（原策略无限制，散户难以兼顾）
  • 单股仓位：≤ 总资金的50%
  • 总仓位：有信号则操作，无信号则空仓
  • 换仓频率：每日清仓，次日重新选股，不隔夜持仓（除非涨停）

6. 风控管理

  • 模式风控：三种模式中只选1–2种执行，避免信息过载
  • 技术风控：5日均线 + 4%硬止损 + 尾盘清仓三重保险
  • 流动性风控：通过成交额（≥1亿）确保可买卖
  • 市值风控：排除极小盘（<30亿）和超级大盘（>600亿）
  • 时间风控：14:50强制检查，杜绝“幻想持仓”

7. 手动实操步骤（纯人工+普通软件版）

第一步：盘后准备（15:30–16:30）

  1. 打开同花顺问财，输入：
    昨日涨停，前日未涨停，非ST，主板或创业板，上市天数>50
  2. 导出结果（约20–50只），保存为“首板观察池”
  3. 对每只股票：
  4. 查F10：确认总市值、流通市值、上市天数
  5. 看K线：确认前日是否阴线（弱转强用）
  6. 记录昨日成交额、60日最高价

第二步：盘前复核（9:15–9:25）

  1. 打开“首板观察池”自选股
  2. 查看集合竞价结果：
  3. 记录每只股开盘涨幅
  4. 估算量比（竞价成交量 vs 昨日9:25时成交量，问财或行情页可见）
  5. 按模式分类：
  6. 低开3%–4.5% + 成交≥1亿 + 60日低位 → 标记A
  7. 高开0–6% + 成交5–20亿 + 量比>3 → 标记B
  8. 前日阴线 + 高开-2%~6% + 成交3–19亿 → 标记C

第三步：开盘买入（9:26–9:35）

  1. 从A/B/C中选出最多2只最符合条件的股票
  2. 计算每只买入金额（总资金 ÷ 股数）
  3. 限价单买入（价格=开盘价），成交后记录成本

第四步：盘中监控（9:35–14:50）

  • 每30分钟检查：
  • 是否跌破5日均线？
  • 是否盈利≥3%后回落？
  • 是否涨停？封单是否稳定？

第五步：尾盘清仓（14:50）

  • 所有未涨停持仓，全部卖出
  • 涨停股若封单减少、有开板迹象，也建议卖出

第六步：盘后复盘（15:00后）

  • 更新交易日志：
    | 日期 | 股票 | 模式 | 买入价 | 卖出价 | 持有天数 | 盈亏% | 成败原因 |
  • 每周统计：各模式胜率、平均收益、最大回撤
  • 连续2日亏损，暂停策略1天

⚠️ 风险提示（散户特别注意）

  • 该策略仍属高风险短线交易，单日亏损可达4%–6%；
  • “弱转强”和“高开模式”对量比、K线判断要求高，新手建议只做“低开模式”；
  • 14:50强制清仓是保命纪律，不可因“可能回封”而犹豫；
  • 强烈建议先用模拟盘或≤10%资金试运行20个交易日，验证执行力后再加大投入。

✅ 一句话总结：
“首板+三选一模式+最多2只+14:50清仓”是散户可行核心。借助问财筛首板池+人工核对开盘数据，普通投资者可有限执行，但必须严守止损与仓位纪律。
