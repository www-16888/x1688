蛇皮走位小市值策略

1.策略的风险等级、分类和策略核心逻辑

  • 风险等级：R4（积极型）
  • 策略分类：小市值策略 / 动量反转策略
  • 策略核心逻辑：通过小市值股票的均值回归特性，结合股价波动率和持仓盈亏状态进行动态仓位调整。核心在于"小市值+低波动+盈利保护"，在控制风险的前提下捕捉小盘股的超额收益。

2.选股条件

  • 选股范围：全市场A股（排除上市未满250天新股）
  • 基础池：全市场正常交易的股票
  • 财务筛选条件：
  • 盈利能力：ROE>15%，ROA>10%
  • 市值要求：总市值从小到大排序
  • 价格要求：股价低于10元
  • 状态排除条件：
  • 排除科创板、北交所股票
  • 排除停牌、ST股票
  • 排除涨停（97%以上）和跌停（104%以下）股票
  • 排序规则：按市值从小到大排序，取前10名

3.买入条件

  • 买入时机：14:30执行买入
  • 买入条件：
  • 波动率条件：股票池波动率方差<0.02且均值>0
  • 盈利条件：持仓总盈利在-10%到40%之间
  • 仓位要求：持仓数量未达到目标数量
  • 买入方式：
  • 等分资金：可用现金平均分配到目标股票
  • 数量限制：最多持有10只股票
  • 顺序买入：按选股顺序依次买入

4.卖出条件

  • 波动止损：波动率条件不满足时卖出非核心持仓
  • 盈利保护：
  • 总盈利超过40%：清仓等待
  • 总亏损超过10%：清仓等待
  • 涨停管理：昨日涨停股票今日涨停打开立即卖出
  • 调仓卖出：不在新选股列表中的股票及时卖出

5.仓位管理

  • 持仓数量：固定持有10只小市值股票
  • 仓位比例：
  • 单一个股：等分资金，分散投资
  • 总仓位：根据波动率和盈利条件动态调整
  • 现金管理：条件不满足时保持现金
  • 动态调整：每日检查，根据市场状态调整仓位

6.风控管理

  • 波动风控：通过波动率方差控制入场时机
  • 盈利风控：严格的盈利保护机制
  • 市值风控：小市值股票的流动性管理
  • 价格风控：低价股筛选避免高价股风险
  • 质量风控：ROE和ROA确保基本面质量

7.手动实操步骤（纯人工交易版）

第一步：盘前准备（9:05） 1. 识别昨日涨停的持仓股票 2. 更新持仓记录和成本数据 3. 准备当日交易计划

第二步：涨停检查（14:00） 1. 检查昨日涨停股票今日表现 2. 涨停打开：立即卖出 3. 继续涨停：保持持有 4. 记录卖出操作和原因

第三步：波动率计算（14:30前） 1. 获取中小板指数成分股 2. 计算流通市值最小的20只股票 3. 计算这些股票近两日的涨跌幅 4. 计算涨跌幅的方差和均值

第四步：盈利状态计算（14:30） 1. 计算每只持仓股票的当前盈亏 2. 汇总持仓总盈利比例 3. 判断是否在-10%到40%的安全区间内

第五步：选股筛选（14:30） 1. 排除上市未满250天的股票 2. 排除科创板、北交所股票 3. 排除停牌、ST、涨跌停股票 4. 筛选股价低于10元的股票 5. 按ROE>15%、ROA>10%进行财务筛选 6. 按市值从小到大排序，取前10名

第六步：交易执行（14:30） 1. 检查波动率条件：方差<0.02且均值>0 2. 检查盈利条件：总盈利在-10%到40%之间 3. 满足条件： - 卖出不在新选股列表中的股票 - 等分资金买入新选出的股票 4. 不满足条件：保持现金或减仓等待

第七步：持仓管理（盘中） 1. 监控持仓股票的价格波动 2. 实时计算总盈利状态 3. 准备应对突发的市场变化

第八步：盘后复盘（15:00后） 1. 分析当日交易执行情况 2. 评估波动率指标的有效性 3. 检查盈利保护机制的触发情况 4. 优化选股参数和交易条件

关键执行要点 - 严格遵循波动率和盈利双重条件 - 专注小市值股票的低估值机会 - 控制持仓数量，避免过度集中 - 及时止盈止损，保护既有收益 - 完善波动率监控体系

风险提示 该策略依赖小市值因子的有效性，在大小盘风格切换时可能表现不佳。波动率条件的设定需要根据市场环境调整，过于严格可能错过机会。小市值股票的流动性风险需要特别注意，大资金操作时可能影响成交价格。
