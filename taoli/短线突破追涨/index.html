短线突破追涨策略（散户实操优化版）

1. 策略分类和核心逻辑

  • 一级分类：技术面短线策略
  • 二级分类：短线突破追涨策略
  • 核心逻辑：聚焦小盘、近期活跃但未近期涨停的股票，在其盘中快速拉升至接近涨停（94.5%–97.5%） 时追涨买入，博弈当日封板或次日冲高。通过多重止盈止损规则+6日持仓上限控制风险，适合有盯盘时间、执行力强的散户捕捉强势股爆发点。

2. 选股条件（适配普通软件）

  • 选股范围：A股主板 + 创业板（排除科创板688/689、北交所4/8开头、ST/*ST、上市<100天新股、停牌股）

  • 筛选方法（全部可通过同花顺问财或通达信实现）：

  • 基础过滤：

    • 非ST、非科创板、非北交所、上市>100天
  • 规模要求：流通市值 < 25亿元（同花顺F10可查）
  • 活跃度筛选：
    • 300天内有涨停（说明有股性）
    • 最近30天无涨停（避免高位接力）
    • 过去15日振幅 > 2.5%（有波动性）

  • 突破形态（盘中动态判断）：

    • 当前价 = 15日内最高价（即创15日新高）
    • 当前价 ≥ 60日最高价 × 92%（接近前高）

  • 更新频率：每日盘前（9:00–9:25）生成候选池，盘中实时监控价格

✅ 同花顺问财语句（盘前筛选用）：
流通市值<25亿，非ST，主板或创业板，上市天数>100，300日内有涨停，30日内无涨停，15日振幅>2.5%，股价>0
（注：突破形态需盘中人工判断，无法完全自动化）

✅ 通达信选股公式（保存为“突破候选池”）：
```pascal { 短线突破追涨 - 候选股池 } NOT_ST := NAMELIKE('ST')=0 AND NAMELIKE('*ST')=0; MAIN_OR_GEM := CODELIKE('60') OR CODELIKE('00') OR CODELIKE('30'); NOT_NEW := FINANCE(42) > 100; LIQ_MCAP := FINANCE(40) < 2500000000; { 流通市值<25亿，单位：元 } HAS_ZT_300D := COUNT(C/REF(C,1)>=1.095,300) >= 1; NO_ZT_30D := COUNT(C/REF(C,1)>=1.095,30) = 0; AMP_15D := (HHV(H,15)-LLV(L,15))/REF(C,15) > 0.025;

NOT_ST AND MAIN_OR_GEM AND NOT_NEW AND LIQ_MCAP AND HAS_ZT_300D AND NO_ZT_30D AND AMP_15D; ```

3. 买入条件

  • 触发信号：盘中实时价格进入涨停价的94.5%–97.5%区间
  • 即：当前价 ∈ [涨停价×0.945, 涨停价×0.975]
  • 例：涨停价10元 → 买入区间为 9.45元 – 9.75元
  • 执行时点：价格首次进入该区间时立即买入（越早越好）
  • 最大持仓：不超过3只（原5只对散户操作压力大，优化为3只）
  • 资金分配：等分可用资金（如总资金9万，买3只则每只3万）
  • 委托方式：市价单或涨停价限价单（确保快速成交，避免踏空）

📌 操作提示：
- 将候选池股票加入自选股分组，开启“价格预警”（同花顺支持设置94.5%–97.5%区间提醒）；
- 若股票快速冲板（如9:45前），可放弃（避免高位接盘）。

4. 卖出条件（简化+可执行）

  • 止盈卖出：
  • 盈利 ≥ 12% → 当日或次日冲高卖出（不强求15%）
  • 盘中冲高后15分钟内回落 ≥ 3% → 视为“尖峰回落”，立即卖出
  • 止损卖出：
  • 亏损 ≥ 6% → 当日或次日开盘止损
  • 持股满6个交易日且盈利 < 5% → 第6日14:50强制卖出
  • 技术卖出：
  • 放量滞涨：当日成交量 ≥ 前5日均量×3，但收盘涨幅 < 2% → 次日卖出
  • 例外：若股票涨停且封单稳定，可持有至开板日卖出（但散户建议次日冲高即卖）

📌 工具支持：
- 同花顺K线图可叠加“成交量”和“15分钟K线”；
- 设置手机闹钟：每日14:50检查持仓。

5. 仓位管理

  • 最大持股数：3只（兼顾操作可行性与分散风险）
  • 单股仓位：≤ 总资金的35%
  • 持股周期：1–6个交易日，目标2–3天
  • 资金使用：滚动操作，卖出后资金当日或次日用于新标的
  • 空仓规则：若无符合买入条件的股票，保持空仓

6. 风控管理

  • 价格风控：只在94.5%–97.5%区间追涨，避免涨停板上追高
  • 时间风控：6日上限防止陷入“假突破”陷阱
  • 技术风控：尖峰回落+放量滞涨双重预警
  • 流动性风控：流通市值<25亿但>5亿（避免极小盘股无法卖出）
  • 情绪风控：涨停股次日若高开低走，不幻想“反包”，果断止盈

7. 手动实操步骤（纯人工+普通软件版）

第一步：盘前准备（8:30–9:00）

  1. 打开同花顺问财，输入筛选语句，生成“突破候选池”（约20–50只）
  2. 将结果导入自选股分组（如命名为“突破监控”）
  3. 对每只股票：
  4. 查F10确认流通市值<25亿
  5. 看K线确认30日内无涨停、15日有波动
  6. 设置价格预警（同花顺APP支持）：
  7. 预警条件：股价 ≥ 涨停价×0.945
  8. 提醒方式：手机推送

第二步：盘中监控（9:30–14:50）

  1. 专注自选股分组，观察价格变动
  2. 一旦某股价格进入94.5%–97.5%区间：
  3. 立即查看：是否放量、是否15日新高、是否接近60日高点
  4. 若符合，9:30–14:30之间快速买入（14:30后不新开仓）
  5. 记录：股票、买入价、买入时间、涨停价

第三步：卖出执行

  • 盘中：若出现尖峰回落（15分钟跌≥3%）或放巨量滞涨，立即卖出
  • 每日14:50：检查所有持仓：
  • 盈利≥12%？→ 卖
  • 持股第6天？→ 卖
  • 亏损≥6%？→ 次日开盘卖
  • 涨停股：次日若高开≤3%，冲高至5%以上卖出；若低开，反弹不过成本即卖

第四步：纪律与记录

  • 每日交易日志：
    | 日期 | 股票 | 买入价 | 涨停价 | 买入区间 | 卖出价 | 持有天数 | 卖出原因 | 盈亏% |
  • 每周复盘：统计成功率、平均收益、失败主因（如“假突破”“尾盘诱多”）
  • 连续2次亏损超5%：暂停策略，检查市场情绪是否退潮

⚠️ 风险提示（散户特别注意）

  • 该策略本质是高风险追涨，在市场情绪退潮期（如连板股大面积跌停）胜率极低，需空仓观望；
  • 94.5%–97.5%区间买入，若当日未封板，次日低开概率高，务必严守止损；
  • 小盘股易被操纵，警惕“尾盘拉升诱多”；
  • 强烈建议先用模拟盘或≤10%资金试运行，熟悉节奏后再加大仓位；
  • 交易成本较高（佣金+印花税+滑点≈0.2%），单笔盈利需>2%才有意义。

✅ 一句话总结：
“小盘+近期活跃+未近期涨停+盘中冲高至94.5%–97.5%”是买入核心，12%止盈、6%止损、6日上限是生存底线。借助问财筛选+价格预警，普通散户可执行，但必须严守纪律。
