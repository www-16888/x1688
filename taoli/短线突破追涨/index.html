短线突破追涨策略

1. 策略分类和核心逻辑

  • 一级分类：技术面短线策略
  • 二级分类：短线突破追涨策略
  • 核心逻辑：通过技术指标筛选即将突破前高的活跃小盘股，在股价接近涨停时追涨买入，结合多种止盈止损条件进行短线交易，追求强势股的短期爆发收益。

2. 选股条件

  • 选股范围：全市场A股
  • 排除范围：
  • 科创板、北交所股票
  • ST股、退市标签股
  • 上市未满100天的新股
  • 当日停牌股票 技术面筛选标准：
  • 规模要求：流通市值 < 25亿元（小盘股偏好）
  • 活跃度筛选：
  • 近期300天内有涨停记录
  • 最近30天内无涨停记录
  • 过去15日内振幅 > 2.5%
  • 突破形态：
  • 股价处于15日内最高点
  • 当前价格接近60日内最高点的92%以上
  • 当前价格在涨停价的94.5%-97.5%之间
  • 买入时机：盘中实时监控，价格达标立即买入

3. 买入条件

  • 交易频率：日内实时交易
  • 买入规则：价格进入涨停价的94.5%-97.5%区间时立即买入
  • 数量限制：最大持有5只股票
  • 资金分配：等分可用资金到每只新买入股票
  • 执行方式：盘中逐笔交易

4. 卖出条件

  • 止盈条件：
  • 收益率 ≥ 15%时卖出
  • 收益率 ≥ 12%且出现尖峰回落时卖出
  • 止损条件：
  • 收益率 ≤ -6%时卖出
  • 持股超过6天且收益率 < 5%时卖出
  • 技术卖出：
  • 放量滞涨：成交量放大3倍但涨幅 < 2%
  • 尖峰回落：15分钟线出现5%以上回落
  • 特殊情况：涨停股票暂不卖出

5. 仓位管理

  • 持仓数量：最大持有5只股票
  • 持股周期：短线持有，最长6天
  • 资金分配：等分可用资金买入
  • 仓位控制：满仓运作，滚动交易
  • 交易频率：日内高频监控

6. 风控管理

  • 价格风控：接近涨停时买入，控制追高风险
  • 时间风控：6天持股期限，避免长期被套
  • 技术风控：多种技术指标联合判断卖出时机
  • 流动性风控：小盘股但控制单只股票仓位
  • 交易风控：涨停保护，避免错过连续涨停

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每个交易日8:30）

  1. 持仓管理：
  2. 更新所有持仓股的持股天数
  3. 记录持股时间变化

第二步：选股准备（每个交易日9:05）

  1. 基础筛选：
  2. 排除科创、北交、ST、次新、停牌股票

  3. 筛选流通市值 < 25亿的小盘股

  4. 技术筛选：

  5. 筛选300天内有涨停历史的活跃股
  6. 排除30天内有涨停的股票（避免追高）
  7. 排除过去15日振幅过小的股票
  8. 筛选处于15日新高的股票
  9. 筛选接近60日新高（92%以上）的股票

第三步：盘中监控（全天交易时间）

  1. 买入监控：
  2. 实时监控选股池股票价格
  3. 当价格进入涨停价的94.5%-97.5%区间时
  4. 立即执行买入操作

  5. 记录买入时间和持股天数

  6. 卖出监控：

  7. 监控持仓股的各种卖出信号
  8. 执行符合条件的卖出操作
  9. 更新持仓信息

第四步：尾盘集中卖出（14:40）

  1. 综合评估：
  2. 检查所有持仓股的卖出条件
  3. 执行尖峰回落、持股超期等卖出
  4. 处理放量滞涨等技术卖出信号

第五步：隔日处理（次日9:30）

  1. 清理残留：
  2. 卖出昨日未卖出的符合条件的股票
  3. 确保持仓清洁

第六步：纪律维护

  1. 实时执行：
  2. 严格执行价格区间的买入纪律
  3. 坚决按技术指标进行卖出

  4. 控制持股时间不超过6天

  5. 风控坚持：

  6. 不因情绪改变买卖标准
  7. 保持小盘股的资金分配均衡
  8. 及时止盈止损，避免亏损扩大

交易成本考虑： - 买入佣金：0.03% - 卖出佣金：0.03% - 印花税：0.1% - 最低佣金：5元/笔 - 滑点：0.03%

风险提示： 该策略为短线追涨策略，交易频率较高，交易成本影响较大。小盘股流动性风险较高，在极端市场情况下可能出现无法及时卖出的情况。追涨策略在震荡市中可能面临较大回撤。多种技术卖出条件可能在某些情况下产生冲突，需要权衡判断。策略对盘中监控要求较高，不适合无法实时盯盘的投资者。
