高股息轮动策略

1. 策略分类和核心逻辑

  • 一级分类：价值投资策略
  • 二级分类：股息率轮动策略
  • 核心逻辑：通过筛选高股息率的优质股票，定期轮动持有，在控制风险的前提下获取稳定的股息收益和资本增值，实现长期稳健回报。

2. 选股条件

  • 选股范围：上海证券交易所 + 深圳证券交易所（主板 + 创业板 + 科创板）
  • 基础筛选：
  • 股票状态：正常股票（排除ST、*ST）
  • 交易状态：非停牌股票
  • 指数成分：属于主要宽基指数成分股
  • 量化筛选：
  • 市值要求：总市值排名前80%
  • 估值要求：市盈率(PE_TTM)排名前40%
  • 盈利要求：市盈率 > 0（排除亏损股）
  • 股价要求：收盘价 ≤ 30元
  • 核心指标：按股息率从高到低排序，选择排名前3的股票

3. 买入条件

  • 调仓频率：每3个交易日调仓一次
  • 买入时点：调仓日上午9:30开盘后立即执行
  • 买入价格：以开盘价买入
  • 买入顺序：
  • 先卖出所有不再符合条件的持仓
  • 再买入新入选的3只股票

4. 卖出条件

  • 定期调仓：每3天评估，不在股息率前3名即卖出
  • 卖出时点：调仓日上午9:30开盘后立即执行
  • 卖出价格：以开盘价卖出
  • 特殊情况：
  • 股票变为ST、*ST
  • 股票停牌超过调仓周期
  • 股票从指数成分股中剔除

5. 仓位管理

  • 持仓数量：固定持有3只股票
  • 仓位分配：每只股票等权重33.33%
  • 总仓位：100%满仓运作
  • 行业分散：自动覆盖不同行业
  • 市值分散：侧重中大市值股票

6. 风控管理

  • 基本面风控：通过指数成分股、盈利要求、估值要求筛选优质标的
  • 流动性风控：选择市值前80%的股票确保流动性
  • 价格风控：30元以下股价控制个股风险
  • 强制轮动：定期调仓避免过度依赖单一个股

7. 手动实操步骤（纯人工交易版）

第一步：调仓日准备（每3个交易日前一晚）

  1. 数据收集：
  2. 获取主要宽基指数成分股名单
  3. 收集股票基本面数据（市值、市盈率、股息率等）

  4. 排除ST、停牌、亏损股票

  5. 初步筛选：

  6. 筛选总市值排名前80%的股票
  7. 筛选市盈率排名前40%且大于0的股票
  8. 筛选股价30元以下的股票

第二步：排序选股（调仓日9:15前）

  1. 计算股息率：
  2. 对初步筛选的股票计算股息率

  3. 按股息率从高到低排序

  4. 确定买入名单：

  5. 选择股息率排名前3的股票
  6. 记录股票代码和名称

第三步：调仓执行（调仓日9:30）

  1. 卖出操作：
  2. 检查当前持仓
  3. 卖出不在前3名单的持仓股

  4. 以开盘价挂单卖出

  5. 买入操作：

  6. 计算可用资金
  7. 等分资金买入3只新股
  8. 以开盘价挂单买入

第四步：日常监控

  1. 状态监控：
  2. 每日检查持仓股是否出现ST风险
  3. 监控停牌情况

  4. 关注指数成分股调整

  5. 数据更新：

  6. 定期更新基本面数据
  7. 为下一次调仓做准备

第五步：特殊情况处理

  1. 紧急卖出：
  2. 持仓股变为ST立即卖出
  3. 股票停牌及时调整策略

  4. 成分股剔除及时处理

  5. 调仓纪律：

  6. 严格按3天周期执行
  7. 坚决执行卖出不再符合条件的股票
  8. 保持等权重配置

交易成本考虑： - 买入费率：0.03% - 卖出费率：0.13% - 最低费用：5元/笔 - 调仓时需考虑交易成本对收益的影响

风险提示：该策略侧重价值投资，适合中长期稳健投资者。股息率受公司分红政策影响，可能存在变化。定期调仓会产生较多交易成本，需在收益预期中充分考虑。市场风格转换可能影响策略表现。
