行业轮动小市值策略（散户实操优化版 · A股合规版）

1. 策略分类和核心逻辑

  • 一级分类：行业轮动策略
  • 二级分类：市场宽度择时 + 小市值选股
  • 核心逻辑：通过同花顺问财自然语言一键筛选基本面优质的小市值股票，结合全市场行业宽度指标排除周期性强的四大行业（银行、有色、钢铁、煤炭），在每周调仓时动态调整持仓。策略简化了行业宽度计算、避免高频交易、降低操作难度，全部条件均可在免费软件中实现，符合A股T+1、100股整数倍、交易时段等规则，适合普通散户每周操作30分钟。

2. 选股条件

  • 选股范围：中证全指成分股（主板+中小板）
  • 排除范围：
  • 创业板（300）、科创板（688/689）、北交所（4/8开头）
  • ST、*ST、退市风险股
  • 停牌股

  • 上市不满375天的新股

  • 行业择时：

  • 行业宽度指标：选择20日均线以上个股比例最高的前N个行业（通常取前3–5个）；
  • 排除四大搅屎棍行业：银行、有色金属、钢铁、煤炭；

  • 空仓机制：若四大行业主导，则保持空仓或持有货币ETF（511880）。

  • 基本面筛选：

  • 盈利能力：ROE > 10%，ROA > 5%（原标准过高，调整为更现实的标准）
  • 市值排序：按总市值从小到大排序，选取市值最小的前10只
  • 技术面筛选：排除停牌股票

✅ 同花顺问财语句（每周一9:00直接复制）：
中证全指成分股，非ST，上市天数大于375，ROE大于10，ROA大于5，总市值从小到大排序

3. 核心筛选流程

第一步：行业分析（每周一9:00–9:15）

  1. 获取行业宽度数据：
  2. 手动计算各行业股价在20日均线以上的比例（可借助通达信、同花顺的板块指数功能）；
  3. 排序生成行业宽度排名；
  4. 若银行、有色、钢铁、煤炭行业占据前几位，则保持空仓或持有货币ETF（511880）。
  5. 确定目标行业：
  6. 选择20日均线以上比例最高的前3–5个行业（排除四大行业）；
  7. 记录目标行业名称。

第二步：选股执行（周一9:15–9:25）

  1. 基础筛选：
  2. 使用上述问财语句筛选符合条件的股票；
  3. 按总市值从小到大排序；
  4. 排除停牌股票。
  5. 行业过滤：
  6. 在问财结果页，点击“所属行业”列，仅保留目标行业内的股票；
  7. 取市值最小的前10只股票（若某行业不足10只，可跨行业补足）。
  8. 最终选择：
  9. 记录买入名单（代码、名称、市值）；
  10. 准备调仓对比名单。

第三步：调仓准备（周一开盘前）

  1. 持仓分析：
  2. 检查当前持仓情况；
  3. 识别昨日涨停的持仓股；
  4. 准备调仓对比名单。
  5. 资金规划：
  6. 计算可用资金总额；
  7. 确定每只新股的买入金额（等分资金）。

4. 买入条件

  • 调仓频率：每周一次（每周一）
  • 持仓数量：固定持有10只股票
  • 资金分配：
  • 可用现金等分买入每只股票；
  • 每只买入数量为100股整数倍。
  • 执行时间：9:35–10:00（避开集合竞价，确保成交）

5. 卖出条件

  • 定期调仓卖出：
  • 每周调仓日，卖出不在新选股名单中的持仓股；
  • 保留仍在名单中的股票（减少交易频率）。
  • 涨停跟踪：
  • 昨日涨停的持仓股，今日涨停打开立即卖出；
  • 若继续涨停则保持持有。
  • 行业择时：
  • 四大搅屎棍行业出现时清仓，转持货币ETF（511880）。
  • 卖出时点：
  • 下午14:00执行涨停股检查；
  • 其他情况下，次日开盘卖出。

6. 仓位管理

  • 持仓数量：固定持有10只股票
  • 资金分配：等权重，每只约占10%资金
  • 总仓位：
  • 若选出≥10只 → 满仓运作；
  • 若选出<10只 → 剩余资金买入货币ETF（511880）。
  • 行业分散：强制覆盖≥3个不同行业（消费、制造、医药等）

7. 风控管理

  • 行业风控：通过排除周期性强的四大行业控制系统性风险；
  • 基本面风控：严格的财务指标筛选确保标的质量；
  • 市值风控：小市值股票但通过基本面过滤控制质量；
  • 流动性风控：排除涨跌停极端情况的股票；
  • 择时风控：行业宽度指标作为市场情绪风控；
  • 止损风控：收盘价连续2日低于20日均线 → 次日开盘卖出。

8. 手动实操步骤

第一步：行业分析（每周一9:00–9:15）

  1. 行业宽度计算：
  2. 获取中证全指成分股；
  3. 计算各行业股价在20日均线以上的比例；
  4. 生成行业宽度排名；
  5. 若银行、有色、钢铁、煤炭行业在领先位置，则保持空仓或持货币ETF。
  6. 搅屎棍检测：
  7. 检查四大行业是否在领先位置；
  8. 若是则保持空仓，否则进入选股流程。

第二步：选股执行（周一9:15–9:25）

  1. 基础筛选：
  2. 排除不符合条件的股票；
  3. 筛选ROE>10%、ROA>5%的股票；
  4. 按总市值从小到大排序。
  5. 最终选择：
  6. 选择前10只市值最小的股票；
  7. 排除停牌和涨跌停股票；
  8. 生成买入名单。

第三步：调仓准备（周一开盘前）

  1. 持仓分析：
  2. 检查当前持仓情况；
  3. 识别昨日涨停的持仓股；
  4. 准备调仓对比名单。
  5. 资金规划：
  6. 计算可用资金总额；
  7. 确定每只新股的买入金额。

第四步：调仓执行（周一9:30）

  1. 卖出操作：
  2. 卖出不在新选股池中的持仓；
  3. 保留昨日涨停且继续涨停的股票。
  4. 买入操作：
  5. 等分资金买入10只新股；
  6. 记录买入价格和基本信息。

第五步：盘中监控（每日14:00）

  1. 涨停股跟踪：
  2. 监控昨日涨停持仓股的表现；
  3. 涨停打开的股票立即卖出；
  4. 记录卖出操作和原因。

第六步：行业监控

  1. 持续观察：
  2. 每日监控行业宽度变化；
  3. 关注四大搅屎棍行业表现；
  4. 及时调整投资策略。
  5. 基本面跟踪：
  6. 定期检查持仓股财务指标；
  7. 关注财报更新对持仓的影响；
  8. 评估策略适应性。

第七步：纪律执行

  1. 周期纪律：
  2. 严格按周度周期调仓；
  3. 坚决执行行业择时规则；
  4. 保持10只股票等权重配置。
  5. 风控纪律：
  6. 严格执行四大行业排除规则；
  7. 及时处理涨停股卖出；
  8. 保持策略一致性。
