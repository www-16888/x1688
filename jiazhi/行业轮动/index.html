行业轮动小市值策略

1. 策略分类和核心逻辑

  • 一级分类：行业轮动策略
  • 二级分类：市场宽度择时 + 小市值选股
  • 核心逻辑：通过监控全市场行业宽度，在排除"四大搅屎棍"行业（银行、有色、钢铁、煤炭）的干扰后，选择基本面优质的小市值股票进行投资，实现行业轮动与小市值溢价的结合。

2. 选股条件

  • 选股范围：中证全指成分股（排除科创板、北交所、创业板、ST股、上市未满375天新股）
  • 行业择时：
  • 计算全市场行业宽度指标
  • 排除银行、有色金属、钢铁、煤炭四大行业主导时期
  • 选择行业宽度表现最好的前N个行业
  • 基本面筛选：
  • 盈利能力：ROE > 15%，ROA > 10%
  • 市值排序：按总市值从小到大排序
  • 技术面筛选：
  • 排除停牌股票

3. 买入条件

  • 调仓频率：每周调仓一次（周一）
  • 买入时点：调仓日上午9:30执行
  • 买入规则：
  • 先卖出不再符合条件的持仓
  • 再买入新入选的股票
  • 数量限制：固定持有10只股票
  • 资金分配：等分可用资金买入

4. 卖出条件

  • 定期调仓：每周评估，不在选股池中即卖出
  • 涨停跟踪：昨日涨停的持仓股，今日涨停打开立即卖出
  • 行业择时：四大搅屎棍行业出现时清仓
  • 卖出时点：下午14:00执行涨停股检查

5. 仓位管理

  • 持仓数量：固定持有10只股票
  • 资金分配：等分资金到每只新买入股票
  • 总仓位：根据行业择时信号动态调整
  • 空仓机制：特定行业出现时保持空仓

6. 风控管理

  • 行业风控：通过排除周期性强的四大行业控制系统性风险
  • 基本面风控：严格的财务指标筛选确保标的质量
  • 市值风控：小市值股票但通过基本面过滤控制质量
  • 流动性风控：排除涨跌停极端情况的股票
  • 择时风控：行业宽度指标作为市场情绪风控

7. 手动实操步骤（纯人工交易版）

第一步：行业分析（每周一9:05前）

  1. 行业宽度计算：
  2. 获取中证全指成分股
  3. 计算各行业股价在20日均线以上的比例

  4. 生成行业宽度排名

  5. 搅屎棍检测：

  6. 检查银行、有色、钢铁、煤炭行业是否在领先位置
  7. 如果出现则保持空仓
  8. 否则进入选股流程

第二步：选股执行（周一9:30前）

  1. 基础筛选：
  2. 排除不符合条件的股票
  3. 筛选ROE>15%、ROA>10%的股票

  4. 按总市值从小到大排序

  5. 最终选择：

  6. 选择前10只市值最小的股票
  7. 排除停牌和涨跌停股票
  8. 生成买入名单

第三步：调仓准备（周一开盘前）

  1. 持仓分析：
  2. 检查当前持仓情况
  3. 识别昨日涨停的持仓股

  4. 准备调仓对比名单

  5. 资金规划：

  6. 计算可用资金总额
  7. 确定每只新股的买入金额

第四步：调仓执行（周一9:30）

  1. 卖出操作：
  2. 卖出不在新选股池中的持仓

  3. 保留昨日涨停且继续涨停的股票

  4. 买入操作：

  5. 等分资金买入10只新股
  6. 记录买入价格和基本信息

第五步：盘中监控（每日14:00）

  1. 涨停股跟踪：
  2. 监控昨日涨停持仓股的表现
  3. 涨停打开的股票立即卖出
  4. 记录卖出操作和原因

第六步：行业监控

  1. 持续观察：
  2. 每日监控行业宽度变化
  3. 关注四大搅屎棍行业表现

  4. 及时调整投资策略

  5. 基本面跟踪：

  6. 定期检查持仓股财务指标
  7. 关注财报更新对持仓的影响
  8. 评估策略适应性

第七步：纪律执行

  1. 周期纪律：
  2. 严格按周度周期调仓
  3. 坚决执行行业择时规则

  4. 保持10只股票等权重配置

  5. 风控纪律：

  6. 严格执行四大行业排除规则
  7. 及时处理涨停股卖出
  8. 保持策略一致性

交易成本考虑： - 买入佣金：0.03% - 卖出佣金：0.03% - 印花税：0.1% - 最低佣金：5元/笔 - 滑点：0%

风险提示：该策略结合行业轮动和小市值选股，波动性相对较大。行业宽度指标可能存在滞后性，需要结合其他指标确认。四大行业排除规则可能在某些市场环境下错过收益机会。小市值股票虽然通过基本面筛选，但仍需注意流动性风险。
