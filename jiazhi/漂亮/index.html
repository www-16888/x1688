漂亮50行业龙头策略（中证500优化版）

1. 策略分类和核心逻辑

  • 一级分类：基本面量化策略
  • 二级分类：多因子选股 + 行业轮动 + 止损风控
  • 核心逻辑：在中证500成分股中，通过多维度基本面因子筛选各行业优质龙头，构建分散化投资组合，结合技术止损控制回撤。

2. 选股条件

  • 选股范围：中证500成分股
  • 核心因子：PEG + 净利润增长率 + ROE + 市值

3. 核心筛选流程

第一步：基础过滤 - 排除范围： - 科创板(688开头) - ST股票 - 退市风险股票 - 上市不足250天 - 市值筛选：剔除流通市值最小的20%

第二步：行业分层筛选 - 行业分类：申万一级行业 - 分层逻辑：每个行业内独立筛选

第三步：多因子筛选 - PEG因子：选择各行业内PEG最低的前70% - 净利润增长率：选择各行业内增长率最高的前70% - ROE因子：选择各行业内ROE最高的前33% - 交集筛选：同时满足三个条件的股票

第四步：市值排序 - 最终排序：按流通市值降序排列 - 逻辑意义：选择行业内的龙头公司

4. 买入条件

  • 调仓周期：10天（双周调仓）
  • 持仓数量：10只股票
  • 资金分配：
  • 可用现金等分买入
  • 动态计算可买数量
  • 执行时间：开盘9:30

5. 卖出条件

  • 调仓卖出：不在新选股列表中的持仓
  • 止损卖出：
  • 计算5日内最高价
  • 当前价较最高价下跌15%时止损
  • 执行时间：尾盘14:55
  • 轮换机制：定期调仓替换

6. 仓位管理

  • 持仓限制：固定10只股票
  • 资金分配：
  • 调仓时等分可用现金
  • 考虑已有持仓情况
  • 风险控制：
  • 行业分散
  • 个股止损

7. 风控管理

  • 质量风控：
  • 严格的股票过滤
  • 基本面因子保障
  • 价格风控：
  • 动态止损机制
  • 避免大幅回撤
  • 执行风控：
  • 排除涨跌停股票
  • 确保交易可行性

8. 交易频率

  • 选股计算：调仓日开盘前
  • 买入执行：9:30开盘
  • 止损监控：14:55尾盘
  • 调仓周期：10个交易日

9. 策略特点

基本面选股优势： - 多维度财务指标 - 行业相对比较 - 龙头公司选择

风险控制完善： - 动态止损保护 - 严格的调仓纪律 - 分散化投资

执行效率： - 明确的买卖规则 - 自动化的调仓流程 - 实时监控机制

10. 手动实操步骤

第一步：数据准备 - 获取中证500成分股 - 执行基础过滤条件 - 准备财务数据

第二步：行业分层 - 按申万一级行业分类 - 各行业内独立筛选 - 确保行业分散

第三步：因子计算 - 计算PEG指标 - 计算净利润增长率 - 计算ROE指标

第四步：筛选排序 - 执行多因子筛选 - 取交集股票 - 按市值排序

第五步：调仓执行 - 检查现有持仓 - 卖出不在列表股票 - 买入新选股票

第六步：止损监控 - 计算持仓最高价 - 监控价格回撤 - 执行止损操作

策略优势： - 科学的基本面选股 - 严格的止损风控 - 行业分散配置 - 长期稳健表现

风险提示：该策略依赖财务数据的准确性和及时性，需关注财报发布周期。建议结合宏观经济环境，注意市场风格切换风险。在价值风格占优的市场中表现较好。
