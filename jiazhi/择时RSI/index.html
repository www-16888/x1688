价值成长RSI择时策略

1. 策略分类和核心逻辑

  • 一级分类：基本面+技术面复合策略
  • 二级分类：价值成长RSI择时策略
  • 核心逻辑：结合价值投资的基本面筛选与RSI技术指标择时，在优质小市值股票中寻找被低估标的，通过月度调仓和日度RSI止盈实现价值发现与技术择时的双重优势。

2. 选股条件

  • 选股范围：全市场A股
  • 排除范围：
  • 科创板、北交所股票
  • ST股及其他退市标签股
  • 上市未满375天的次新股
  • 当日停牌股票

基本面筛选标准： - 估值指标： - 市净率（PB）：0-10倍 - 市现率（PCF）：< 5倍 - 盈利能力： - 净资产收益率（ROE）：> 30% - 销售毛利率：> 30% - 成长能力： - 净利润同比增长率：> 150% - 市值偏好： - 按总市值从小到大排序 - 初步筛选50只最小市值股票

技术面筛选标准： - RSI指标： - 14日RSI < 50（处于弱势区间） - 选择被低估的优质股票 - 最终持仓：选择前10只股票构建组合

3. 买入条件

  • 交易频率：月度调仓（每月第1个交易日）
  • 买入规则：
  • 通过基本面筛选50只候选股
  • 用RSI技术指标进一步筛选
  • 等权重分配资金到最终10只股票
  • 执行方式：开盘立即执行调仓

4. 卖出条件

  • 定期调仓：月度统一调仓，卖出非目标股
  • 技术止盈：
  • 每日检查持仓股RSI指标
  • 14日RSI > 70时卖出（超买区域）
  • 及时锁定盈利，避免回调
  • 调仓逻辑：完全根据月度选股结果决定去留

5. 仓位管理

  • 持仓数量：固定10只股票
  • 资金分配：严格的等权重分配
  • 调仓频率：月度调仓，降低交易频率
  • 监控频率：每日技术指标监控
  • 仓位控制：接近满仓运作

6. 风控管理

  • 基本面风控：严格的财务指标确保标的质量
  • 估值风控：控制估值水平，避免过高溢价
  • 技术风控：RSI指标及时止盈，控制回撤
  • 分散风控：10只股票分散投资，降低个股风险
  • 流动性风控：排除高风险板块股票

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每个交易日9:05）

  1. 持仓管理：
  2. 更新当前持仓列表
  3. 准备当日监控工作

第二步：月度选股（每月第1个交易日）

  1. 基础筛选：
  2. 获取全市场股票列表

  3. 排除科创、北交、ST、次新、停牌股票

  4. 基本面筛选：

  5. 筛选PB在0-10倍、PCF<5倍的股票
  6. 筛选ROE>30%、毛利率>30%的股票
  7. 筛选净利润增长>150%的股票

  8. 按市值排序选择前50只最小市值股票

  9. 技术面筛选：

  10. 计算50只候选股的14日RSI值
  11. 选择RSI < 50的股票（处于弱势但基本面优质）
  12. 取前10只构建最终投资组合

第三步：月度调仓（每月第1个交易日9:30）

  1. 调仓执行：
  2. 卖出所有不在新组合中的持仓
  3. 等分资金买入10只目标股票
  4. 完成月度调仓

第四步：每日监控（每个交易日9:30）

  1. 技术止盈：
  2. 计算每只持仓股的14日RSI值
  3. 如RSI > 70，立即卖出该股票
  4. 实现及时止盈，锁定收益

第五步：持续优化

  1. 参数回顾：
  2. 回顾基本面筛选标准的有效性
  3. 优化RSI参数的敏感性
  4. 调整持仓数量与资金分配

策略逻辑解析： - 价值投资逻辑：寻找高ROE、高增长、合理估值的好公司 - 小市值效应：利用小市值股票的历史超额收益 - 逆向思维：在RSI弱势时买入，避免追高 - 趋势跟踪：在RSI超买时卖出，及时止盈 - 复合优势：基本面确保质量，技术面优化时机

交易成本考虑： - 买入佣金：0.03% - 卖出佣金：0.03% - 印花税：0.1% - 最低佣金：5元/笔 - 滑点：0.3%

风险提示： 该策略对财务数据的准确性有较高依赖，需要关注财报发布周期的影响。RSI指标在趋势性行情中可能过早卖出，错过更大涨幅。小市值股票在流动性紧张时可能面临较大的冲击成本。高ROE和高增长的要求可能限制可选标的范围，在特定市场环境下可选股票较少。策略在价值因子失效时期可能表现不佳。
