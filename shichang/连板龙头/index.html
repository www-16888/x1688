连板龙头策略

1. 策略分类和核心逻辑

  • 一级分类：事件驱动策略
  • 二级分类：连板龙头股策略
  • 核心逻辑：通过识别市场中的连板龙头股票，在情绪周期上升阶段参与最强标的，利用龙头股的溢价效应获取超额收益，同时通过严格的仓位控制和卖出条件管理风险。

2. 选股条件

  • 选股范围：全市场A股（排除科创板、北交所、ST股、停牌股、上市未满50天新股）
  • 筛选方法：
  • 基础过滤：排除不符合条件的股票
  • 涨停识别：筛选当日涨停股票
  • 连板计算：计算每只股票的连续涨停天数
  • 龙头确定：选择连板天数最高的股票作为龙头
  • 核心指标：按5日平均换手率排序，选择换手率最小的标的

3. 买入条件

  • 买入时机：上午9:30开盘后立即执行
  • 买入方式：
  • 开盘未涨停：使用市价单立即买入
  • 开盘涨停：使用限价单排板买入
  • 排序规则：当出现多只龙头股时，选择换手率最小的标的
  • 数量限制：最多持有10只股票

4. 卖出条件

  • 卖出条件1：当日不涨停
  • 卖出条件2：持有超过2个交易日
  • 卖出条件3：已实现盈利
  • 执行时点：下午14:50执行卖出
  • 卖出原则：满足条件1的前提下，条件2或条件3任一满足即卖出

5. 仓位管理

  • 分仓数量：最多同时持有10只股票
  • 资金分配：等分总资金到每只持仓股
  • 仓位控制：单只股票仓位不超过总资金的10%
  • 持仓限制：确保持股数量不超过分仓上限

6. 风控管理

  • 流动性风控：通过换手率筛选避免流动性风险
  • 价格风控：涨停板买入限制追高风险
  • 时间风控：2天持有期限控制风险暴露时间
  • 分散风控：多股票持仓分散单一标的风险
  • 强制卖出：不涨停即卖出的纪律性风控

7. 手动实操步骤（纯人工交易版）

第一步：盘前准备（每个交易日9:01前）

  1. 数据收集：
  2. 获取前一日涨停股票列表
  3. 计算每只股票的连续涨停天数

  4. 排除ST、次新、停牌股票

  5. 龙头识别：

  6. 找出连板天数最高的股票
  7. 记录候选龙头股名单

第二步：开盘决策（9:15-9:30）

  1. 换手率排序：
  2. 计算候选股的5日平均换手率

  3. 按换手率从小到大排序

  4. 最终选择：

  5. 选择换手率最小的龙头股
  6. 确保总持仓不超过10只

第三步：买入执行（9:30）

  1. 买入判断：
  2. 观察开盘是否涨停
  3. 未涨停：立即市价买入

  4. 已涨停：限价单排板

  5. 仓位控制：

  6. 计算单股买入金额：总资金/10
  7. 确保不超过仓位限制

第四步：盘中监控

  1. 持仓跟踪：
  2. 记录每只股票的买入日期

  3. 监控是否继续涨停

  4. 盈利计算：

  5. 实时计算持仓盈亏
  6. 为卖出决策做准备

第五步：卖出执行（14:50）

  1. 卖出条件检查：
  2. 检查持仓股是否涨停
  3. 计算持有天数

  4. 计算当前盈亏

  5. 卖出决策：

  6. 不涨停的股票进入卖出流程
  7. 持有超2天或已盈利的股票卖出
  8. 严格执行卖出纪律

第六步：盘后复盘

  1. 持仓整理：
  2. 更新持仓记录
  3. 计算总体收益

  4. 准备次日交易计划

  5. 策略优化：

  6. 分析交易记录
  7. 调整选股参数
  8. 优化买卖时机

风险提示：该策略属于高风险高收益类型，连板股波动性极大，可能出现连续跌停无法卖出的风险。需要严格执行止损纪律，控制单票仓位，避免过度集中。市场情绪转换时策略可能面临较大回撤。
