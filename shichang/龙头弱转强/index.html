龙头弱转强战法（散户观察版 · 低频简化）

1. 策略分类和核心逻辑

  • 一级分类：技术面波段策略（非超短线）
  • 二级分类：龙头股回调低吸策略
  • 核心逻辑：
    不追“跌停次日高开”的高风险信号，而是等待龙头股经历短期调整（非跌停）后，出现“缩量企稳 + 放量阳线” 的弱转强确认信号，次日低吸，持有3–5天。
    从“搏反弹”变为“等确认”，牺牲部分爆发力，换取更高胜率和可操作性。

2. 选股条件（人工可查）

✅ 选股范围：

  • 主板A股（60/00开头），排除ST、*ST、科创板、创业板
  • 上市满180天（避免次新波动）
  • 近20日涨幅 > 20%（确认为近期龙头）

✅ 调整阶段特征（非跌停！）：

  • 最近3日内出现单日跌幅 ≥ 7%（但未跌停）
  • 调整期间缩量：下跌日成交量 < 前5日均量
  • 未破20日均线：股价仍在20日均线上方（趋势未坏）

3. 买入信号（次日确认）

✅ 弱转强确认信号（需同时满足）：

  1. 出现中阳线：当日涨幅 ≥ 3%，且收盘价 > 前日最高价
  2. 放量回升：成交量 > 前5日均量
  3. 非高位：当前价 < 近20日最高价的95%（避免追高）
  4. 开盘不追：次日（T+1）若高开 ≤ 3%，可在10:00后回踩不破昨日收盘价时低吸

💡 操作节奏：
- T日：大跌缩量
- T+1日：放量阳线（观察日，不买）
- T+2日：若高开≤3%且分时企稳 → 买入

4. 买入与卖出规则

买入：

  • 时间：T+2日 10:00–10:30（避开开盘情绪化波动）
  • 仓位：单只 ≤ 总资金20%，最多同时持2只
  • 方式：100股整数倍，不追高（若高开>5%则放弃）

卖出：

  • 止盈：
  • 持有3天后，若未达目标（如+8%），则止盈
  • 若涨停 → 持有至开板或次日冲高卖出
  • 止损：
  • 买入价 -6% 无条件止损
  • 跌破买入日最低价 → 止损

5. 手动实操步骤（无需盯盘）

每日收盘后（15:30–16:00）：

  1. 打开同花顺，筛选“近20日涨幅>20%”的主板股票
  2. 查看是否有股票当日跌幅≥7%但未跌停
  3. 检查是否缩量 + 未破20日线 → 加入“观察池”

次日（T+1）收盘后：

  1. 检查观察池股票是否出现放量阳线（+3%以上）
  2. 若符合 → 标记为“候选”，准备T+2日操作

T+2日：

  • 若高开 ≤ 3%，且10:00后股价不破昨日收盘价 → 买入
  • 否则放弃，等待下一次机会

6. 通达信 / 问财友好支持

✅ 问财选股语句（收盘后用）：

主板，非ST，近20日涨幅>20%，今日跌幅>7%且跌幅<9.9%，成交量<5日均量，股价>20日均线，上市满180天

次日验证语句：

主板，非ST，昨日跌幅>7%，今日涨幅>3%，今日成交量>5日均量，股价<20日最高价*0.95

7. 给想尝试短线的散户建议

  1. 先用模拟盘练习1个月，验证策略有效性
  2. 单笔风险 ≤ 总资金2%（如10万账户，单只止损≤2000元）
  3. 只做大盘环境好时（上证>3000且5日均线上行）
  4. 优先选择热门板块龙头（如AI、新能源、消费电子）
  5. 绝不补仓：短线失败即离场
