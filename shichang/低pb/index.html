低PB小市值低换手策略（散户简化版）

1. 策略分类和核心逻辑

  • 一级分类：价值投资 + 市场有效性
  • 二级分类：低估值 + 小市值 + 低关注度
  • 核心逻辑：
    每月从全市场A股中，选择低市净率（PB）、小市值、低换手率的股票，通过分散投资、定期调仓，捕捉市场过度反应带来的价值回归机会。

2. 选股条件（人工可查）

✅ 选股范围：

  • 主板A股（60/00开头），排除科创板（688）、创业板（300）
  • 非ST、*ST股票（财务状况异常）
  • 正常交易状态（非停牌、非涨停）

✅ 核心指标：

条件 标准 查看位置
低市净率（PB） PB < 1.5倍 同花顺F10 → “主要财务指标”或行情界面
小市值 总市值 < 100亿元 F10 → “公司概况” 或 行情界面
低换手率 近5日平均换手率 < 1% 行情界面或同花顺“个股资料”中的“换手率”

💡 为什么选这些阈值？
- PB < 1.5倍：通常认为低于1.5倍的股票具有一定的估值安全边际。
- 市值 < 100亿：避免过大市值导致成长性不足，同时控制流动性风险。
- 换手率 < 1%：表明市场关注度较低，可能被低估。

3. 核心筛选流程（每月第一个交易日）

第一步：基础数据获取

  • 数据字段：市值、市净率、换手率、股票代码
  • 数据时点：调仓日前一个交易日

第二步：双重筛选机制

筛选路径1：低PB + 低换手

  1. 按PB升序排列，选取前20只
  2. 在低PB股票中，按换手率升序排列，选取前10只

筛选路径2：小市值 + 低换手

  1. 按市值升序排列，选取前20只
  2. 在小市值股票中，按换手率升序排列，选取前10只

第三步：合并去重

  • 合并两个候选池，去除重复股票，最终得到不超过10只的目标股票

4. 买入条件

✅ 买入规则：

  • 调仓时间：每月第一个交易日 9:30
  • 持仓数量：固定 10只股票
  • 买入方式：
  • 等权重分配（总资金 ÷ 10）
  • 优先选择行业分散的股票（避免全买银行或全买地产）
  • 若某只股票涨停无法买入，则跳过，从备选中补足10只

5. 卖出条件

✅ 卖出规则：

  • 常规卖出：调仓日卖出不在新名单中的股票
  • 强制卖出：不符合新选股标准的持仓全部卖出

6. 仓位管理

✅ 仓位控制：

  • 总持仓：通常满仓（除非大盘极度高估）
  • 单个股票：10%（严格等权）
  • 调仓频率：每月1次（降低摩擦成本）
  • 行业分散：尽量覆盖3个以上行业（如银行、电力、交运、化工、公用事业等）

7. 风控管理

✅ 风险控制：

  • 质量风控：排除ST、*ST股票，确保财务健康
  • 流动性风控：排除停牌股票，确保可正常交易
  • 价格风控：排除涨停股票，确保可买入
  • 情绪风控：不追热点、不炒题材、不碰“看不懂”的公司

8. 手动实操步骤（纯人工版）

📅 每月第一个交易日操作流程（9:00–9:30）

第一步：用问财快速初筛（9:00–9:10）

在 同花顺i问财（https://www.iwencai.com）输入：

主板，非ST，PB<1.5，市值<100亿，近5日平均换手率<1%，上市满1年

✅ 结果通常为50~100只股票，导出或截图。

第二步：人工精筛（9:10–9:25）

对问财结果，逐个检查F10： 1. 是否连续盈利？（近3年至少有盈利）
2. 是否有大额负债或亏损？（看“主要财务指标”）
3. 所属行业是否稳健？（优先选：银行、电力、高速、港口、煤炭、家电等）
4. 当日是否可交易？（非涨跌停、非停牌）

选出10只，按“行业分散 + PB最低 + 换手率最低”排序。

第三步：调仓执行（9:30）

  1. 卖出：不在新名单的持仓（涨停股除外）
  2. 买入：等分资金买入新10只
  3. 记录：在Excel登记股票、成本、行业、PB、换手率

9. 通达信 / 同花顺友好支持

✅ 问财选股语句（直接复制）：

主板，非ST，PB<1.5，市值<100亿，近5日平均换手率<1%，上市满1年

10. 策略特点与优势

✅ 策略特点：

  • 明确的估值信号：低PB提供安全边际
  • 严格的筛选机制：多重过滤确保标的质量
  • 简单的执行流程：每月一次调仓，适合上班族
  • 历史表现稳健：在价值风格占优的市场中表现较好

✅ 优势：

  • 低风险：低PB提供估值保护，小市值具备成长潜力
  • 高胜率：多维度筛选，减少“价值陷阱”概率
  • 易操作：无需高频盯盘，适合普通投资者

❗ 风险提示：

  • 价值陷阱：部分低PB股票可能存在基本面恶化
  • 波动较大：小市值股票波动性较大，需心理准备
  • 市场风格轮动：在成长风格主导的市场中可能跑输
