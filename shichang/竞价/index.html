集合竞价量比策略

1. 策略分类和核心逻辑

  • 一级分类：技术面短线策略
  • 二级分类：集合竞价 + 量比选股 + 高换手突破
  • 核心逻辑：在集合竞价阶段捕捉量比异常放大且换手率高的股票，开盘后立即买入，尾盘卖出，实现日内短线交易。

2. 选股条件

  • 选股范围：全市场A股（排除科创板）
  • 核心指标：量比 + 集合竞价换手率

3. 核心筛选流程

第一步：基础过滤 - 排除范围： - 科创板(688开头) - ST股票 - 停牌股票 - 退市股票 - 上市时间：上市满180天 - 确保流动性：正常交易状态

第二步：量比筛选 - 量比计算：5日平均成交量比率 - 排序方式：按量比降序排列 - 选择数量：前10名股票

第三步：换手率筛选 - 换手计算： - 集合竞价成交额 / 流通市值 - 流通市值单位：亿元 - 换手门槛：换手率 > 0.3% - 排序方式：按换手率降序排列

4. 买入条件

  • 时间窗口：9:27（开盘前）
  • 仓位控制：
  • 最大持仓：3只股票
  • 资金分配：可用现金等分
  • 动态调整：根据持仓数量
  • 执行逻辑：
  • 持仓数 < 3时买入
  • 按换手率排序选择
  • 等金额分配资金

5. 卖出条件

  • 卖出时间：14:45（尾盘）
  • 卖出条件：
  • 非涨停状态可卖出
  • 清空所有持仓
  • 持仓周期：日内交易（T+0）

6. 仓位管理

  • 持仓限制：最多3只股票
  • 资金分配：
  • 可用现金等分买入
  • 动态计算可买数量
  • 风险控制：
  • 分散持仓
  • 日内了结

7. 风控管理

  • 质量风控：
  • 排除问题股票
  • 确保交易流动性
  • 执行风控：
  • 涨停限制卖出
  • 避免无效交易
  • 时间风控：
  • 严格的买卖时间
  • 日内了结风险

8. 交易频率

  • 选股计算：9:26（集合竞价）
  • 买入执行：9:27（开盘前）
  • 卖出执行：14:45（尾盘）
  • 持仓周期：单日持有

9. 策略特点

量价分析优势： - 量比识别资金关注 - 高换手确认活跃度 - 集合竞价数据利用

短线交易特点： - 日内了结规避隔夜风险 - 严格的买卖时间控制 - 明确的止损止盈机制

执行效率： - 自动化的选股流程 - 快速的交易执行 - 清晰的仓位管理

10. 手动实操步骤

第一步：盘前准备 - 获取全市场股票数据 - 执行基础过滤条件 - 准备数据计算

第二步：集合竞价分析 - 计算量比指标 - 筛选前10名股票 - 获取集合竞价数据

第三步：换手率计算 - 计算流通市值 - 计算集合竞价换手率 - 筛选高换手股票

第四步：买入执行 - 检查持仓数量 - 计算可买股票数量 - 等金额分配买入

第五步：盘中监控 - 监控持仓状态 - 准备尾盘卖出

第六步：尾盘卖出 - 检查涨停状态 - 清空所有持仓 - 记录交易信息

策略优势： - 明确的量价突破逻辑 - 严格的日内风险控制 - 高效的交易执行 - 适应短线交易特点

风险提示：该策略属于高风险短线策略，集合竞价数据波动性大。建议结合大盘环境，注意流动性风险，严格执行止损纪律。在活跃市场中效果较好，震荡市中需谨慎使用。
