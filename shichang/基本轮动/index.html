基本面小市值轮动策略

1. 策略分类和核心逻辑

  • 一级分类：价值投资策略
  • 二级分类：基本面小市值轮动策略
  • 核心逻辑：通过筛选基本面优质的小市值股票，结合月度轮动和涨停板跟踪，在控制风险的前提下获取小市值溢价和基本面改善带来的超额收益。

2. 选股条件

  • 选股范围：全市场A股（排除科创板、北交所、ST股、停牌股、上市未满250天新股）
  • 基本面筛选：
  • 盈利能力：ROE > 15%，ROA > 10%
  • 成长性：净利润同比增长率 > 0
  • 估值水平：PB < 3
  • 技术面筛选：
  • 价格限制：股价 < 10元
  • 涨跌幅限制：排除当日涨幅过大（接近涨停）和跌幅过大股票
  • 核心排序：按总市值从小到大排序，选择市值最小的股票

3. 买入条件

  • 调仓频率：每月调仓一次
  • 买入时点：调仓日上午9:30执行
  • 买入规则：
  • 先卖出不再符合条件的持仓
  • 再买入新入选的股票
  • 数量限制：固定持有10只股票
  • 黑名单机制：排除最近30天内持有过且出现涨停的股票

4. 卖出条件

  • 定期调仓：每月评估，不在选股池中即卖出
  • 涨停跟踪：昨日涨停的持仓股，今日涨停打开立即卖出
  • 特殊情况：4月份强制清仓（空仓期）
  • 卖出时点：调仓日下午14:55执行

5. 仓位管理

  • 持仓数量：固定持有10只股票
  • 资金分配：等分可用资金到每只新买入股票
  • 总仓位：100%满仓运作
  • 行业分散：通过全市场选股自动实现行业分散

6. 风控管理

  • 基本面风控：严格的财务指标筛选确保标的质量
  • 市值风控：小市值股票但排除微盘股风险
  • 价格风控：10元以下股价控制个股风险
  • 流动性风控：排除涨跌停极端情况的股票
  • 时间风控：月度轮动避免过度交易，4月空仓规避风险

7. 手动实操步骤（纯人工交易版）

第一步：月度选股（每月最后一个交易日）

  1. 基础筛选：
  2. 排除ST、次新、停牌股票
  3. 筛选股价10元以下的股票

  4. 排除接近涨跌停的股票

  5. 财务数据收集：

  6. 获取ROE、ROA、PB等财务指标
  7. 筛选ROE>15%、ROA>10%、PB<3的股票

  8. 按总市值从小到大排序

  9. 黑名单过滤：

  10. 检查最近30天持有过且出现涨停的股票
  11. 从候选池中排除这些股票
  12. 选择前10只市值最小的股票

第二步：调仓准备（调仓日9:05）

  1. 持仓分析：
  2. 检查当前持仓
  3. 识别昨日涨停的持仓股

  4. 准备调仓名单

  5. 资金规划：

  6. 计算可用资金
  7. 确定每只新股的买入金额

第三步：盘中监控（调仓日14:00）

  1. 涨停股跟踪：
  2. 监控昨日涨停持仓股的表现
  3. 涨停打开的股票立即卖出
  4. 记录卖出操作

第四步：调仓执行（调仓日14:55）

  1. 卖出操作：
  2. 卖出不在新选股池中的持仓

  3. 执行调仓卖出指令

  4. 买入操作：

  5. 等分资金买入10只新股
  6. 确保总持仓数为10只

第五步：特殊情况处理

  1. 4月清仓：
  2. 4月份强制清仓所有持仓
  3. 保持空仓状态

  4. 为5月重新建仓做准备

  5. 日常监控：

  6. 每日检查持仓股基本面变化
  7. 监控财务指标是否持续符合要求
  8. 关注市场风格变化

第六步：纪律维护

  1. 月度复盘：
  2. 分析月度收益情况
  3. 检查策略执行效果

  4. 优化选股参数

  5. 严格执行：

  6. 坚决按月度周期调仓
  7. 严格执行4月清仓规则
  8. 保持10只股票等权重配置

交易成本考虑： - 买入佣金：0.03% - 卖出佣金：0.03% - 印花税：0.1% - 最低佣金：5元/笔 - 滑点：0.02%

风险提示：该策略侧重小市值股票，虽然通过基本面筛选控制质量，但小市值股票波动性相对较大。4月空仓期可能错过上涨行情，需要权衡风险收益。财务数据存在滞后性，需关注财报更新对持仓的影响。
