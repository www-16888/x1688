分时量比突增策略（散户手动实操版）

一、策略分类

  • 策略类型：短线T+1博弈（隔夜持仓）
  • 适用市场环境：
  • 震荡市或情绪平稳期；
  • 板块轮动启动日或突发利好发酵日（如政策、行业新闻）；
  • 非退潮期（高位股无批量跌停）
  • 核心逻辑：
    个股在盘中出现明显放量拉升（量比突增的简化版），且处于板块前列，若尾盘能稳住涨幅，则次日有冲高溢价。

二、选股条件（T日盘中观察 + 尾盘确认）

第一步：早盘预筛观察池（可选）

在同花顺问财输入（T日早盘前）：

所属行业近1日有政策利好，或行业指数近3日涨幅前10，非ST，流通市值30–200亿

💡 或直接关注同花顺“行业涨幅榜”前3的板块，从中选个股。

第二步：T日盘中人工观察（13:30–14:30）

从涨幅榜或自选股中，筛选同时满足以下条件的个股：

条件 判断方式（同花顺免费版）
1. 盘中明显放量拉升 分时图出现陡峭上涨，且量柱突然显著放大（肉眼可见，至少是上午平均量柱的2倍以上）
2. 拉升幅度 ≥ 3% 5分钟内快速上涨 ≥ 3%（非缓慢推升）
3. 处于当日高位 当前价接近当日最高价（分时图高点附近，未大幅回落）
4. 所属行业强势 所属行业当日涨幅排名前3（同花顺“行业板块”页面可查）

三、买入条件（T日 14:45–14:55）

仅当上述4项条件满足，且尾盘稳住涨幅时，方可买入：

  • ✅ 股价在14:45时仍处于当日高点附近（回落 ≤ 1%）；
  • ✅ 成交量下午明显高于上午（全天呈“放量上涨”结构）；
  • ✅ 所属行业仍居涨幅前列（未大幅回落）；
  • ❌ 若14:45前已大幅回落（如从+5%跌至+1%），或行业排名下滑 → 放弃。

📌 买入方式：
- 14:45–14:55市价单手动买入；
- 单票仓位 ≤ 10%（因信号偏情绪驱动，不确定性高）。

四、卖出条件（T+1日）

⚠️ A股T+1规则：T日买入，T+1日9:30起可卖出

✅ 止盈：

  • 高开 ≥ 2%，冲高不破T日高点 → 9:45–10:15分批卖出；
  • 平开后快速冲高（+3%以上）→ 冲高回落破分时均线即卖；
  • 若继续强势涨停 → 持有，但开板即清仓。

❌ 止损：

  • 低开 ≤ -1.5% 且30分钟内无法翻红 → 止损；
  • 反弹不过T日收盘价 → 止损；
  • 所属行业T+1日领跌 → 优先离场。

📌 原则：T+1日必须清仓，不持仓过周末。

五、仓位管理

项目 建议
单票仓位 ≤ 10%
每日最多操作 1只 股票
每周操作频率 ≤ 2次（高质量信号稀少）
空仓条件 - 大盘单边下跌
- 行业轮动混乱（无明确领涨板块）
- 市场情绪低迷（连板高度≤2）

六、手动实操步骤（每日流程）

时间 操作 工具
T日早盘 关注“行业涨幅榜”前3板块，加入相关个股到自选股 同花顺“行业板块”
T日 10:00–14:30 留意自选股是否出现“放量陡升+行业强势” 分时图 + 成交量柱
T日 14:45 确认：是否稳在高位？量能是否持续？行业是否仍强？ 同花顺APP
T日 14:45–14:55 信号成立 → 手动买入；否则放弃 券商交易软件
T+1日 9:30–10:30 严格执行止盈/止损，必须卖出 —
每周复盘 记录：是否真放量？行业是否持续？次日溢价如何？ 笔记本/Excel

⚠️ 重要提醒

  • 该策略依赖板块轮动节奏，若市场无主线，成功率骤降；
  • 避免追高已涨5%以上的个股，只做“放量启动初期”；
  • 小市值+行业利好组合效果最佳，但需警惕“一日游”行情。
