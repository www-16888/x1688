N字反攻突破策略

1、选股条件

前提：聚焦“趋势股”或“题材活跃股”

步骤1：建立观察池（T日早盘前）

在同花顺问财输入：

近5日涨幅>5%，非ST，非北交所，所属概念板块近3日有涨停，流通市值50–300亿

💡 或直接从“昨日涨停未连板”“5日均线向上”个股中筛选。

步骤2：T日盘中人工确认（需同时满足）

条件 判断方式（同花顺APP）
1. 早盘有冲高 10:00前出现明显高点（H），涨幅 ≥ 3%（分时图可见陡升）
2. 长时间横盘 10:00–13:30 股价在 H点下方2%以内 震荡（肉眼观察未大幅跌破）
3. 午后放量突破 13:30后股价放量突破H点，且量柱明显高于上午（肉眼可见“突然放大”）
4. 突破时未暴涨 突破H点时，当日总涨幅 < 7%（留有次日溢价空间）
5. 板块未退潮 所属板块当日未领跌，最好有其他个股跟涨

2、买入条件

仅当上述5项条件满足，且尾盘站稳H点上方时，方可买入：

  • ✅ 14:45时股价仍高于H点（未假突破后回落）；
  • ✅ 下午成交量显著高于上午（全天呈“早冲高+午蓄势+尾放量”结构）；
  • ✅ 所属板块保持活跃（未集体跳水）；
  • ❌ 若14:45前已跌破H点，或突破后无量 → 放弃。

📌 买入方式：
- 14:45–14:55市价单手动买入；
- 单票仓位 ≤ 15%（因涉及突破，需防范假信号）。

💡 若有底仓：可于突破时T+0做T（高抛低吸），但本策略主逻辑面向无底仓散户。

3、卖出条件

⚠️ A股T+1规则：T日买入，T+1日9:30起可卖出

✅ 止盈：

  • 高开 ≥ 2%，冲高不破T日高点 → 9:45–10:15分批卖出；
  • 平开后快速冲高（+3%以上）→ 冲高回落破分时均线即卖；
  • 若T+1日涨停 → 持有，但开板立即清仓。

❌ 止损：

  • 低开 ≤ -2% 且30分钟内无法翻红 → 止损；
  • 反弹不过T日H点（早盘高点） → 止损；
  • 所属板块T+1日走弱 → 优先离场。

📌 原则：T+1日必须清仓，不赌连板（除非是龙头且情绪极强）。

4、仓位管理

项目 建议
单票仓位 ≤ 15%
每日最多操作 1只 股票
每周操作频率 ≤ 2次（高质量N字结构不多）
空仓条件 - 大盘单边下跌
- 题材无持续性（一日游）
- 个股处于下降趋势

5、手动实操步骤（每日流程）

时间 操作 工具
T日早盘前 用问财筛选趋势/题材股，加入自选股 问财 + 同花顺
T日 9:30–10:00 关注自选股是否早盘冲高（记下H点位置） 分时图
T日 10:00–13:30 观察是否在H点附近横盘（不破位） 分时图
T日 13:30–14:30 留意是否放量突破H点 分时图 + 成交量柱
T日 14:45 确认：是否站稳H点？量能是否持续？板块是否强？ 同花顺APP
T日 14:45–14:55 信号成立 → 手动买入；否则放弃 券商交易软件
T+1日 9:30–10:30 严格执行止盈/止损，必须卖出 —
每周复盘 记录：H点是否有效？突破是否放量？次日溢价如何？ 笔记本
